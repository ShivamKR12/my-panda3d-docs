{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classTexture}{Texture}}}]{ }\end{DoxyParamCaption})}

This method is called after each texture has been loaded from disk, via the \doxylink{classTexturePool}{Texture\+Pool}, for the first time. By the time this method is called, the \doxylink{classTexture}{Texture} has already been fully read from disk. This method should return the \doxylink{classTexture}{Texture} pointer that the \doxylink{classTexturePool}{Texture\+Pool} should actually return (usually it is the same as the pointer supplied).
