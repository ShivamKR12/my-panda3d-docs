\subsubsection*{template$<$class Cycle\+Data\+Type$>$\newline
class Cycle\+Data\+Writer$<$ Cycle\+Data\+Type $>$}
This template class calls Pipeline\+Cycler\+::write() in the constructor and Pipeline\+Cycler\+::release\+\_\+write() in the destructor. In the interim, it provides a transparent read-\/write access to the \doxylink{classCycleData}{Cycle\+Data}.

It exists as a syntactic convenience to access the data in the \doxylink{classCycleData}{Cycle\+Data}. It also allows the whole system to compile down to nothing if DO\+\_\+\+PIPELINING is not defined.
