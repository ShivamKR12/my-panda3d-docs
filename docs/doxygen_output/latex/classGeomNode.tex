\doxysubsection{Detailed Description}
A node that holds \doxylink{classGeom}{Geom} objects, renderable pieces of geometry. This is the primary kind of leaf node in the scene graph; almost all visible objects will be contained in a \doxylink{classGeomNode}{Geom\+Node} somewhere.
