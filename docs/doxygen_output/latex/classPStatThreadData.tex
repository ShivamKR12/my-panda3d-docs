\doxysubsection{Detailed Description}
A collection of Frame\+Data structures for recently-\/received frames within a particular thread. This holds the raw data as reported by the client, and it automatically handles frames received out-\/of-\/order or skipped. You can ask for a particular frame by frame number or time and receive the data for the nearest frame.
