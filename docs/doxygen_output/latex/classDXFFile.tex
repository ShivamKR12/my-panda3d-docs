\doxysubsection{Detailed Description}
A generic DXF-\/reading class. This class can read a DXF file but doesn\textquotesingle{}t actually do anything with the data; it\textquotesingle{}s intended to be inherited from and the appropriate functions overridden (particularly Done\+Entity()).
