\doxysubsection{Detailed Description}
This defines a bounding convex hexahedron. It is typically used to represent a frustum, but may represent any enclosing convex hexahedron, including simple boxes. However, if all you want is an axis-\/aligned bounding box, you may be better off with the simpler \doxylink{classBoundingBox}{Bounding\+Box} class.
