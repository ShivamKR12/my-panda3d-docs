\doxysubsection{Detailed Description}
Defines a single collision event. One of these is created for each collision detected by a \doxylink{classCollisionTraverser}{Collision\+Traverser}, to be dealt with by the \doxylink{classCollisionHandler}{Collision\+Handler}.

A \doxylink{classCollisionEntry}{Collision\+Entry} provides slots for a number of data values (such as intersection point and normal) that might or might not be known for each collision. It is up to the handler to determine what information is known and to do the right thing with it.
