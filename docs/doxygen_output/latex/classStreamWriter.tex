\doxysubsection{Detailed Description}
A \doxylink{classStreamWriter}{Stream\+Writer} object is used to write sequential binary data directly to an ostream. Its interface is very similar to \doxylink{classDatagram}{Datagram} by design; it\textquotesingle{}s primarily intended as a convenience to eliminate the overhead of writing bytes to a \doxylink{classDatagram}{Datagram} and then writing the \doxylink{classDatagram}{Datagram} to a stream.
