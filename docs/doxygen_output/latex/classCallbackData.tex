\doxysubsection{Detailed Description}
This is a generic data block that is passed along to a \doxylink{classCallbackObject}{Callback\+Object} when a callback is made. It contains data specific to the particular callback type in question.

This is actually an abstract base class and contains no data. Specializations of this class will contain the actual data relevant to each callback type.
