class Config\+Variable\+Enum$<$ Enum\+Type $>$}
This class specializes \doxylink{classConfigVariable}{Config\+Variable} as an enumerated type. It is a template class, so it cannot be easily published; it\textquotesingle{}s not really necessary outside of C++ anyway.

This variable assumes that the enumerated type in question has input and output stream operators defined that do the right thing (outputting a sensible string for the type, and converting a string to the correct value).
