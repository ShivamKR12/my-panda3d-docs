\doxysubsection{Detailed Description}
A \doxylink{classStencilAttrib}{Stencil\+Attrib} is a collection of all stencil render states. The render states in a \doxylink{classStencilAttrib}{Stencil\+Attrib} are read-\/only. A \doxylink{classStencilAttrib}{Stencil\+Attrib} is created with make or make\+\_\+2\+\_\+sided. To determine if two sided stencil is supported, call the function \doxylink{classGraphicsStateGuardian}{Graphics\+State\+Guardian}\+:\+: get\+\_\+supports\+\_\+two\+\_\+sided\+\_\+stencil.
