\doxysubsection{Detailed Description}
This is a base class for queries that might require a round-\/trip to the graphics engine. The idea is that when you ask the GSG to make a particular query, it returns a \doxylink{classQueryContext}{Query\+Context}, which does not necessarily have the answer right away (but it will eventually).

Unlike \doxylink{classSavedContext}{Saved\+Context}, \doxylink{classQueryContext}{Query\+Context} is reference-\/counted. It removes itself from the GSG when the last reference goes away. You\textquotesingle{}re responsible for keeping the pointer to the \doxylink{classQueryContext}{Query\+Context} as long as you are interested in the answer.
