\doxysubsection{Detailed Description}
This is the non-\/template part of the base class for \doxylink{classPointerTo}{Pointer\+To} and \doxylink{classConstPointerTo}{Const\+Pointer\+To}. It is necessary so we can keep a pointer to a non-\/template class within the \doxylink{classReferenceCount}{Reference\+Count} object, to implement weak reference pointers--we need to have something to clean up when the \doxylink{classReferenceCount}{Reference\+Count} object destructs.

This is the base class for Pointer\+To\+Base$<$\+T$>$.
