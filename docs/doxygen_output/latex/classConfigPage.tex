\doxysubsection{Detailed Description}
A page of Config\+Declarations that may be loaded or unloaded. Typically this represents a single .prc file that is read from disk at runtime, but it may also represent a list of declarations built up by application code and explicitly loaded.

{\footnotesize\ttfamily void Config\+Page\+::set\+\_\+sort (\begin{DoxyParamCaption}\item[{int}]{sort }\end{DoxyParamCaption})}

Changes the explicit sort order of this particular \doxylink{classConfigPage}{Config\+Page}. Lower-\/ numbered pages supercede higher-\/numbered pages. Initially, all explicitly-\/ loaded pages have sort value 0, and implicitly-\/loaded pages (found on disk) have sort value 10; you may set an individual page higher or lower to influence its priority relative to other pages.
