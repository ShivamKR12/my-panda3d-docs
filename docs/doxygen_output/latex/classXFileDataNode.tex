\doxysubsection{Detailed Description}
This is an abstract base class for an \doxylink{classXFileNode}{XFile\+Node} which is also an \doxylink{classXFileDataObject}{XFile\+Data\+Object}. That is to say, objects that inherit from this class may be added to the toplevel X file graph as nodes, and they also may be containers for data elements.

Specifically, this is the base class of both \doxylink{classXFileDataNodeTemplate}{XFile\+Data\+Node\+Template} and \doxylink{classXFileDataNodeReference}{XFile\+Data\+Node\+Reference}.
