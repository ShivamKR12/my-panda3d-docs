\doxysubsection{Detailed Description}
This is the actual interface to a particular VRPN button device, and all of its numbered buttons. A pointer to this object is stored in the \doxylink{classVrpnClient}{Vrpn\+Client} class for each differently-\/named VRPN button device we connect to.

The VRPN callbacks go here, which in turn get vectored out to any \doxylink{classVrpnButtonDevice}{Vrpn\+Button\+Device} objects that register with this. When the last \doxylink{classVrpnButtonDevice}{Vrpn\+Button\+Device} object unregisters, the \doxylink{classVrpnButton}{Vrpn\+Button} will be deleted by the \doxylink{classVrpnClient}{Vrpn\+Client}.

This class does not need to be exported from the DLL.
