\doxysubsection{Detailed Description}
This class handles threaded delivery of datagrams to various TCP or UDP sockets.

A \doxylink{classConnectionWriter}{Connection\+Writer} may define an arbitrary number of threads (0 or more) to write its datagrams to sockets. The number of threads is specified at construction time and cannot be changed.
