\doxysubsection{Detailed Description}
This is a base class for a generic data structure that can be attached per-\/ instance to the camera, to store per-\/instance data that must be preserved over multiple frames.

In particular, this is used to implement the \doxylink{classFadeLODNode}{Fade\+LODNode}, which must remember during traversal at what point it is in the fade, separately for each instance and for each camera.
