\doxysubsection{Detailed Description}
This is the base class for a number of special render effects that may be set on scene graph nodes to change the way they render. This includes \doxylink{classBillboardEffect}{Billboard\+Effect}, \doxylink{classDecalEffect}{Decal\+Effect}, etc.

\doxylink{classRenderEffect}{Render\+Effect} represents render properties that must be applied as soon as they are encountered in the scene graph, rather than propagating down to the leaves. This is different from \doxylink{classRenderAttrib}{Render\+Attrib}, which represents properties like color and texture that don\textquotesingle{}t do anything until they propagate down to a \doxylink{classGeomNode}{Geom\+Node}.

You should not attempt to create or modify a \doxylink{classRenderEffect}{Render\+Effect} directly; instead, use the make() method of the appropriate kind of effect you want. This will allocate and return a new \doxylink{classRenderEffect}{Render\+Effect} of the appropriate type, and it may share pointers if possible. Do not modify the new \doxylink{classRenderEffect}{Render\+Effect} if you wish to change its properties; instead, create a new one.
