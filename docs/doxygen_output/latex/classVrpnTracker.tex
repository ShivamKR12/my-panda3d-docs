\doxysubsection{Detailed Description}
This is the actual interface to a particular VRPN tracker object, and all of its sensors. A pointer to this object is stored in the \doxylink{classVrpnClient}{Vrpn\+Client} class for each differently-\/named VRPN tracker we connect to.

The VRPN callbacks go here, which in turn get vectored out to any \doxylink{classVrpnTrackerDevice}{Vrpn\+Tracker\+Device} objects that register with this. When the last \doxylink{classVrpnTrackerDevice}{Vrpn\+Tracker\+Device} object unregisters, the \doxylink{classVrpnTracker}{Vrpn\+Tracker} will be deleted by the \doxylink{classVrpnClient}{Vrpn\+Client}.

This class does not need to be exported from the DLL.
