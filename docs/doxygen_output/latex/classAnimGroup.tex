\doxysubsection{Detailed Description}
This is the base class for \doxylink{classAnimChannel}{Anim\+Channel} and \doxylink{classAnimBundle}{Anim\+Bundle}. It implements a hierarchy of Anim\+Channels. The root of the hierarchy must be an \doxylink{classAnimBundle}{Anim\+Bundle}.
