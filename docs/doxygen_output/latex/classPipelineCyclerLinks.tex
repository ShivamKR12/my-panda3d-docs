\doxysubsection{Detailed Description}
This just stores the pointers to implement a doubly-\/linked list of Pipeline\+Cyclers for a particular \doxylink{classPipeline}{Pipeline} object. We use a hand-\/rolled linked list rather than any STL container, because we want Pipeline\+Cyclers to be able to add and remove themselves from this list very quickly.

These pointers are inherited from this separate class so the \doxylink{classPipeline}{Pipeline} object itself can be the root of the linked list.
