\subsubsection*{template$<$class Cycle\+Data\+Type$>$\newline
class Cycle\+Data\+Stage\+Writer$<$ Cycle\+Data\+Type $>$}
This class is similar to \doxylink{classCycleDataWriter}{Cycle\+Data\+Writer}, except it allows writing to a particular stage of the pipeline. Usually this is used to implement writing directly to an upstream pipeline value, to recompute a cached value there (otherwise, the cached value would go away with the next pipeline cycle).
