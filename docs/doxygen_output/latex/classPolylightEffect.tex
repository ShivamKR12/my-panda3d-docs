\doxysubsection{Detailed Description}
A \doxylink{classPolylightEffect}{Polylight\+Effect} can be used on a node to define a Light\+Group for that node. A Light\+Group contains Polylight\+Nodes which are essentially nodes that add color to the polygons of a model based on distance. \doxylink{classPolylightNode}{Polylight\+Node} is a cheap way to get lighting effects specially for night scenes
