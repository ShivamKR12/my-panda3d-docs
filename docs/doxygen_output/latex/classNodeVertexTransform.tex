\doxysubsection{Detailed Description}
This \doxylink{classVertexTransform}{Vertex\+Transform} gets its matrix from the Transform stored on a node. It can also compose its node\textquotesingle{}s transform with another \doxylink{classVertexTransform}{Vertex\+Transform}, allowing you to build up a chain of Node\+Vertex\+Transforms that represent a list of composed matrices.
