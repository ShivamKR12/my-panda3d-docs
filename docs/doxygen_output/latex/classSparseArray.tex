\doxysubsection{Detailed Description}
This class records a set of integers, where each integer is either present or not present in the set.

It is similar in principle and in interface to a \doxylink{classBitArray}{Bit\+Array} (which can be thought of as a set of integers, one integer corresponding to each different bit position), but the \doxylink{classSparseArray}{Sparse\+Array} is implemented as a list of min/max subrange lists, rather than as a bitmask.

This makes it particularly efficient for storing sets which consist of large sections of consecutively included or consecutively excluded elements, with arbitrarily large integers, but particularly inefficient for doing boolean operations such as \& or \texorpdfstring{$\vert$}{|}.

Also, unlike \doxylink{classBitArray}{Bit\+Array}, the \doxylink{classSparseArray}{Sparse\+Array} can store negative integers.
