\doxysubsection{Detailed Description}
This is the data for one array of a \doxylink{classGeomVertexData}{Geom\+Vertex\+Data} structure. Many \doxylink{classGeomVertexData}{Geom\+Vertex\+Data} structures will only define one array, with all data elements interleaved (DirectX 8.\+0 and before insisted on this format); some will define multiple arrays.

DirectX calls this concept of one array a "{}stream"{}. It also closely correlates with the concept of a vertex buffer.

This object is just a block of data. In general, you should not be directly messing with this object from application code. See \doxylink{classGeomVertexData}{Geom\+Vertex\+Data} for the organizing structure, and see Geom\+Vertex\+Reader/\+Writer/\+Rewriter for high-\/level tools to manipulate the actual vertex data.
