\doxysubsection{Detailed Description}
This object uses the Freetype library to generate text directly into an image. It is different from the Text\+Node/\+Dynamic\+Text\+Font interface, which use the Freetype library to generate text in the scene graph, to be rendered onscreen via the Panda render traversal.
