class indirect\+\_\+equals\+\_\+hash$<$ Key $>$}
An STL function object class, this is intended to be used on any ordered collection of pointers to classes that contain an operator ==() method. It defines the equality of the pointers via operator ==().

Since it doesn\textquotesingle{}t define the ordering of the pointers, it can only be used with hash containers.
