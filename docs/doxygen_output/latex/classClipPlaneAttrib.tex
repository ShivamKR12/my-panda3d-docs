\doxysubsection{Detailed Description}
This functions similarly to a \doxylink{classLightAttrib}{Light\+Attrib}. It indicates the set of clipping planes that modify the geometry at this level and below. A \doxylink{classClipPlaneAttrib}{Clip\+Plane\+Attrib} can either add planes or remove planes from the total set of clipping planes in effect.

\doxysubsubsection{\texorpdfstring{compare\_to\_impl()}{compare\_to\_impl()}}
{\footnotesize\ttfamily virtual int Clip\+Plane\+Attrib\+::compare\+\_\+to\+\_\+impl (\begin{DoxyParamCaption}\item[{const \mbox{\hyperlink{classRenderAttrib}{Render\+Attrib}} \texorpdfstring{$\ast$}{*}}]{other }\end{DoxyParamCaption}) const\hspace{0.3cm}{\ttfamily [protected]}, {\ttfamily [virtual]}}

Intended to be overridden by derived \doxylink{classRenderAttrib}{Render\+Attrib} types to return a unique number indicating whether this \doxylink{classRenderAttrib}{Render\+Attrib} is equivalent to the other one.
