\doxysubsection{Detailed Description}
This effect will be added automatically to a node by Character\+Joint\+::add\+\_\+net\+\_\+transform() and \doxylink{classCharacterJoint_a69a3b595b723f17ce985310ff5c40fe4}{Character\+Joint\+::add\+\_\+local\+\_\+transform()}.

The effect binds the node back to the character, so that querying the relative transform of the affected node will automatically force the indicated character to be updated first.
