\doxysubsection{Detailed Description}
This object performs a depth-\/first traversal of the scene graph, with optional view-\/frustum culling, collecting Cull\+State and searching for Geom\+Nodes. Each renderable \doxylink{classGeom}{Geom} encountered is passed along with its associated \doxylink{classRenderState}{Render\+State} to the \doxylink{classCullHandler}{Cull\+Handler} object.
