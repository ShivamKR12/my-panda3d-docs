\doxysubsection{Detailed Description}
This represents a "{}layer"{} as read from the DXF file. A layer may be defined by reading the header part of the file, or it may be implicitly defined by an entity\textquotesingle{}s having referenced it.

User code may derive from \doxylink{classDXFLayer}{DXFLayer} to associate private data with each layer, if desired.
