\doxysubsection{Detailed Description}
This is a special node that instances its contents using a list of transforms that get applied on top of the node\textquotesingle{}s own transform. This is a bit more limited than the regular instance\+\_\+to mechanism, but it is a better choice for hardware instancing.

For best performance, it is highly recommended to flatten the nodes under this (by calling flatten\+\_\+strong()), since culling will not be performed for individual sub-\/nodes under each instance.
