\doxysubsection{Detailed Description}
This object represents a window on the desktop, not necessarily a Panda window. This structure can be assigned to a \doxylink{classWindowProperties}{Window\+Properties} to indicate a parent window.

It also has callbacks so the Panda window can communicate with its parent window, which is particularly important when running embedded in a browser.

To create a \doxylink{classWindowHandle}{Window\+Handle}, you would usually call one of the Native\+Window\+Handle\+::make\+\_\+\texorpdfstring{$\ast$}{*}() methods, depending on the kind of native window handle object you already have.
