\doxysubsection{Detailed Description}
A collection of Geoms and their associated state, for a particular scene. The cull traversal (and the \doxylink{classBinCullHandler}{Bin\+Cull\+Handler}) assigns Geoms to bins as it comes across them.

This is an abstract base class; derived classes like \doxylink{classCullBinStateSorted}{Cull\+Bin\+State\+Sorted} and \doxylink{classCullBinBackToFront}{Cull\+Bin\+Back\+To\+Front} provide the actual implementation.
