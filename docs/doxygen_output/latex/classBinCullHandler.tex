\doxysubsection{Detailed Description}
This \doxylink{classCullHandler}{Cull\+Handler} sends all of the geoms it receives into a \doxylink{classCullResult}{Cull\+Result} object, for binning (and later drawing). This is the kind of \doxylink{classCullHandler}{Cull\+Handler} to use for most normal rendering needs.
