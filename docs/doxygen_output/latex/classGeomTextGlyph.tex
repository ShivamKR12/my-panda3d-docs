\doxysubsection{Detailed Description}
This is a specialization on \doxylink{classGeom}{Geom} for containing a primitive intended to represent a \doxylink{classTextGlyph}{Text\+Glyph}. Its sole purpose is to maintain the geom count on the glyph, so we can determine the actual usage count on a dynamic glyph (and thus know when it is safe to recycle the glyph).
