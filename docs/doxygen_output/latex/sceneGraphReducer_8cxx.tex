{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classPandaNode}{Panda\+Node}}}]{ }\end{DoxyParamCaption})}

Collapses the two nodes into a single node, if possible. The \textquotesingle{}siblings\textquotesingle{} flag is true if the two nodes are siblings nodes; otherwise, node1 is a parent of node2. The return value is the resulting node, which may be either one of the source nodes, or a new node altogether, or it may be NULL to indicate that the collapse operation could not take place.
