\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Type$>$\newline
class Deleted\+Chain$<$ Type $>$}
This template class can be used to provide faster allocation/deallocation for many Panda objects. It works by maintaining a linked list of deleted objects that are all of the same type; when a new object is allocated that matches that type, the same space is just reused.

This class is actually a layer on top of \doxylink{classDeletedBufferChain}{Deleted\+Buffer\+Chain}, which handles the actual allocation. This class just provides the typecasting.

Of course, this trick of maintaining the deleted object chain won\textquotesingle{}t work in the presence of polymorphism, where you might have many classes that derive from a base class, and all of them have a different size--unless you instantiate a \doxylink{classDeletedChain}{Deleted\+Chain} for {\itshape every} kind of derived class. The ALLOC\+\_\+\+DELETED\+\_\+\+CHAIN macro, below, is designed to make this easy.
