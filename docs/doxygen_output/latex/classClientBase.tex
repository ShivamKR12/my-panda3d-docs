\doxysubsection{Detailed Description}
An abstract base class for a family of client device interfaces--including trackers, buttons, dials, and other analog inputs.

This provides a common interface to connect to such devices and extract their data; it is used by \doxylink{classTrackerNode}{Tracker\+Node} etc. to put these devices in the data graph.
