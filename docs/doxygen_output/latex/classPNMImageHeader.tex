\doxysubsection{Detailed Description}
This is the base class of \doxylink{classPNMImage}{PNMImage}, \doxylink{classPNMReader}{PNMReader}, and \doxylink{classPNMWriter}{PNMWriter}. It encapsulates all the information associated with an image that describes its size, number of channels, etc; that is, all the information about the image except the image data itself. It\textquotesingle{}s the sort of information you typically read from the image file\textquotesingle{}s header.
