\doxysubsection{Detailed Description}
This class is used during joint optimization or restructuring to store the table of interim joint computations.

That is to say, this class provides an temporary data store for three tables of matrices per each \doxylink{classEggJointPointer}{Egg\+Joint\+Pointer} per frame.
