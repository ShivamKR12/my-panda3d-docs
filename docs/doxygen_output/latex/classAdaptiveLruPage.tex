\doxysubsection{Detailed Description}
One atomic piece that may be managed by a \doxylink{classAdaptiveLru}{Adaptive\+Lru} chain. To use this class, inherit from it and override evict\+\_\+lru().

This class multiply inherits from two classes which in turn both inherit from \doxylink{classLinkedListNode}{Linked\+List\+Node}. This is just a sneaky C++ trick to allow this class to inherit from \doxylink{classLinkedListNode}{Linked\+List\+Node} twice, so that pages can be stored on two different linked lists simultaneously. The \doxylink{classAdaptiveLru}{Adaptive\+Lru} class depends on this; it maintains its pages in two different lists, one grouped by priority, and one in order by next partial update needs.
