\doxysubsection{Detailed Description}
A base class for things which need to inherit from both \doxylink{classTypedWritable}{Typed\+Writable} and from \doxylink{classReferenceCount}{Reference\+Count}. It\textquotesingle{}s convenient to define this intermediate base class instead of multiply inheriting from the two classes each time they are needed, so that we can sensibly pass around pointers to things which are both Typed\+Writables and Reference\+Counters.

See also \doxylink{classTypedObject}{Typed\+Object} for detailed instructions.
