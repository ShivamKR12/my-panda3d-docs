\doxysubsection{Detailed Description}
Corresponding to an $<$\+Xfm\$\+Anim$>$ entry, this stores a two-\/dimensional table with up to nine columns, one for each component of a transformation. This is an older syntax of egg anim table, not often used currently--it\textquotesingle{}s replaced by \doxylink{classEggXfmSAnim}{Egg\+Xfm\+SAnim}.
