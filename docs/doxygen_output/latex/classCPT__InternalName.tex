\doxysubsection{Detailed Description}
This is a const pointer to an \doxylink{classInternalName}{Internal\+Name}, and should be used in lieu of a CPT(\+Internal\+Name) in function arguments. The extra feature that it offers is that it has a constructor to automatically convert from a string, so that strings are coerced by the compiler when passed to such a function.
