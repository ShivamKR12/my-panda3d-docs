\doxysubsection{Detailed Description}
Simple directed vector force. This force is different from the others in that it can be global and still only affect a single object. That might not make sense for a physics simulation, but it\textquotesingle{}s very handy for a game. I.\+e. this is the force applied by user on the selected object.
