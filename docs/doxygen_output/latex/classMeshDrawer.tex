\doxysubsection{Detailed Description}
Mesh drawer creates a single geom object that can be shaped with different draw commands. This is an efficient way to render bunch of billboards, particles, fast changing triangles. Its implemented by recycling same geom over and over again. Max budget specifies how many triangles are allowed. Some uses of this class can be \+: particle system, radar icons, health bars, 2d icons, 2d ui, bullets, missile trails. Any that can be drawn with triangles can be drawn with this class. At the low level this uses the \doxylink{classGeomVertexRewriter}{Geom\+Vertex\+Rewriter}\textquotesingle{}s. The internal geom consists of vertex, normal, uv and color channels.
