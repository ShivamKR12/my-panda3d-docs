\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Type$>$\newline
class Param\+Value$<$ Type $>$}
A handy class object for storing simple values (like integers or strings) passed along with an \doxylink{classEvent}{Event} or to be used as a shader input. This is essentially just a wrapper around whatever data type you like, to make it a \doxylink{classTypedWritableReferenceCount}{Typed\+Writable\+Reference\+Count} object which can be passed along inside an \doxylink{classEventParameter}{Event\+Parameter} or \doxylink{classShaderInput}{Shader\+Input}.
