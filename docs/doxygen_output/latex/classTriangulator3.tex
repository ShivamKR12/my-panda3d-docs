\doxysubsection{Detailed Description}
This is an extension of \doxylink{classTriangulator}{Triangulator} to handle polygons with three-\/ dimensional points. It assumes all of the points lie in a single plane, and internally projects the supplied points into 2-\/D for passing to the underlying \doxylink{classTriangulator}{Triangulator} object.
