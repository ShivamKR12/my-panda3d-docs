{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classGeomPrimitive}{Geom\+Primitive}}}]{ }\end{DoxyParamCaption})}

Returns the fundamental rendering type of this primitive\+: whether it is points, lines, or polygons.

This is used to set up the appropriate antialiasing settings when Antialias\+Attrib\+::\+M\+\_\+auto is in effect; it also implies the type of primitive that will be produced when decompose() is called. Calls the appropriate method on the GSG to draw the primitive. Tells the \doxylink{classBamReader}{Bam\+Reader} how to create objects of type \doxylink{classGeom}{Geom}.
