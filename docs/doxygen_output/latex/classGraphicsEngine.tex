\doxysubsection{Detailed Description}
This class is the main interface to controlling the render process. There is typically only one \doxylink{classGraphicsEngine}{Graphics\+Engine} in an application, and it synchronizes rendering to all all of the active windows; although it is possible to have multiple \doxylink{classGraphicsEngine}{Graphics\+Engine} objects if multiple synchronicity groups are required.

The \doxylink{classGraphicsEngine}{Graphics\+Engine} is responsible for managing the various cull and draw threads. The application simply calls engine-\/\texorpdfstring{$>$}{>}render\+\_\+frame() and considers it done.
