Sorts matrices lexicographically, componentwise. Returns a number less than 0 if this matrix sorts before the other one, greater than zero if it sorts after, 0 if they are equivalent (within the indicated tolerance). Sets mat to a matrix that rotates by the given angle in degrees counterclockwise about the indicated vector.

Fills mat with a matrix that rotates by the given angle in degrees counterclockwise about the indicated vector. Assumes axis has been prenormalized.

Returns true if two matrices are memberwise equal within a specified tolerance. This is faster than the equivalence operator as this doesn\textquotesingle{}t have to guarantee that it is transitive. Adds the vector to the indicated hash generator.

Writes the matrix to the \doxylink{classDatagram}{Datagram} using add\+\_\+float32() or add\+\_\+float64(), depending on the type of floats in the matrix, regardless of the setting of Datagram\+::set\+\_\+stdfloat\+\_\+double(). This is appropriate when you want to write a fixed-\/width value to the datagram, especially when you are not writing a bam file.
