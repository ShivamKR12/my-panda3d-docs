\doxysubsection{Detailed Description}
This funny bounding volume is an infinite line with no thickness and extending to infinity in both directions.

Note that it {\itshape always} extends in both directions, despite the fact that you specify two points to the constructor. These are not endpoints, they are two arbitrary points on the line.
