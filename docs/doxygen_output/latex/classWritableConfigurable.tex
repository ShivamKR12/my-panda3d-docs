\doxysubsection{Detailed Description}
Defined as a fix to allow creating \doxylink{classConfigurable}{Configurable} and Writable objects. Otherwise the compiler gets confused since both \doxylink{classTypedWritable}{Typed\+Writable} and \doxylink{classConfigurable}{Configurable} inherit from \doxylink{classTypedObject}{Typed\+Object}.

An object that has data or parameters that are set less frequently (at least occasionally) than every frame. We can cache the configuration info by by using the "{}dirty"{} flag.
