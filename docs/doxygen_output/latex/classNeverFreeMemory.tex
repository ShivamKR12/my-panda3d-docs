\doxysubsection{Detailed Description}
This class is used to allocate bytes of memory from a pool that is never intended to be freed. It is particularly useful to support \doxylink{classDeletedChain}{Deleted\+Chain}, which allocates memory in just such a fashion.

When it is known that memory will not be freed, it is preferable to use this instead of the standard malloc() (or global\+\_\+operator\+\_\+new()) call, since this will help reduce fragmentation problems in the dynamic heap. Also, memory allocated from here will exhibit less wasted space.
