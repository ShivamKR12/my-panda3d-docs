\doxysubsection{Detailed Description}
This is the bare functionality (intended to be inherited from along with \doxylink{classProgramBase}{Program\+Base} or some derivative) for a program that might generate an output file.

This provides the \doxylink{classWithOutputFile_acd5039b8e878fefc32d7ecc780e4cfa9}{has\+\_\+output\+\_\+filename()} and \doxylink{classWithOutputFile_a4c2299fef0437b67a951f4f7a3ccf937}{get\+\_\+output\+\_\+filename()} methods.

This is called when the output file is given as the last parameter on the command line. Since this is a fairly dangerous way to specify the output file (it\textquotesingle{}s easy to accidentally overwrite an input file this way), the convention is to disallow this syntax if the output file already exists.

This function will test if the output file exists, and issue a warning message if it does, returning false. If all is well, it will return true.
