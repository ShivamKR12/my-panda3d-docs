\doxysubsection{Detailed Description}
This class can be used to write a series of datagrams into a memory buffer. It acts as both a datagram sink and generator; you can fill it up with datagrams and then read as many datagrams from it.

This uses the same format as \doxylink{classDatagramInputFile}{Datagram\+Input\+File} and \doxylink{classDatagramOutputFile}{Datagram\+Output\+File}, meaning that \doxylink{classDatagram}{Datagram} sizes are always stored little-\/endian.
