\doxysubsection{Detailed Description}
The abstract base class for all things that can collide with other things in the world, and all the things they can collide with (except geometry).

This class and its derivatives really work very similarly to the way \doxylink{classBoundingVolume}{Bounding\+Volume} and all of its derivatives work. There\textquotesingle{}s a different subclass for each basic shape of solid, and double-\/dispatch function calls handle the subset of the N\texorpdfstring{$\ast$}{*}N intersection tests that we care about.
