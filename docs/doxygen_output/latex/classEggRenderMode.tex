\doxysubsection{Detailed Description}
This class stores miscellaneous rendering properties that is associated with geometry, and which may be set on the geometry primitive level, on the group above it, or indirectly via a texture. It\textquotesingle{}s intended to be a base class for egg objects that can have these properties set.

This class cannot inherit from \doxylink{classEggObject}{Egg\+Object}, because it causes problems at the \doxylink{classEggPolygon}{Egg\+Polygon} level with multiple appearances of the \doxylink{classEggObject}{Egg\+Object} base class. And making \doxylink{classEggObject}{Egg\+Object} a virtual base class is just no fun.
