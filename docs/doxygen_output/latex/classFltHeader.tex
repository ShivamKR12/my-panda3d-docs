\doxysubsection{Detailed Description}
This is the first bead in the file, the top of the bead hierarchy, and the primary interface to reading and writing a Flt file. You always read a Flt file by creating a header and calling \doxylink{classFltHeader_a6afae3c73850ac375442dd40ebfcd4ab}{read\+\_\+flt()}, which fills in its children beads automatically; you write a Flt file by creating a header, adding its children, and calling \doxylink{classFltHeader_ab42cafbb8cb335755b27db8c9a4e8474}{write\+\_\+flt()}.
