\doxysubsection{Detailed Description}
This is a special type of \doxylink{classPStatTimer}{PStat\+Timer} that also uses a timer query on the GSG to measure how long a task actually takes to execute on the GPU, rather than how long it took for the API commands to be queued up.

This class may only be used on the draw thread.

At present, it tracks both the CPU time (like a regular \doxylink{classPStatTimer}{PStat\+Timer} does) and the GPU time, which is recorded using a special \doxylink{classPStatThread}{PStat\+Thread}.
