\doxysubsection{Detailed Description}
A class that encapsulates the extra bytes that are sent in front of each datagram to identify it when it is sent on TCP. This is similar to \doxylink{classDatagramUDPHeader}{Datagram\+UDPHeader}, except it does not include a checksum, since this is unnecessary on UDP.
