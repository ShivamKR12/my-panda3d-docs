\doxysubsection{Detailed Description}
The primary interface to this module. This class does basic text assembly; given a string of text and a \doxylink{classTextFont}{Text\+Font} object, it creates a piece of geometry that may be placed in the 3-\/d or 2-\/d world to represent the indicated text.

The \doxylink{classTextNode}{Text\+Node} may be used in one of two ways. Naively, it may simply be parented directly into the scene graph and rendered as if it were a \doxylink{classGeomNode}{Geom\+Node}; in this mode, the actual polygon geometry that renders the text is not directly visible or accessible, but remains hidden within the \doxylink{classTextNode}{Text\+Node}.

The second way \doxylink{classTextNode}{Text\+Node} may be used is as a text generator. To use it in this way, do not parent the \doxylink{classTextNode}{Text\+Node} to the scene graph; instead, set the properties of the text and call generate() to return an ordinary node, containing ordinary geometry, which you may use however you like. Each time you call generate() a new node is returned.
