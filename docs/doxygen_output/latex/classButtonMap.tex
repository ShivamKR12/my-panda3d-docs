\doxysubsection{Detailed Description}
This class represents a map containing all of the buttons of a (keyboard) device, though it can also be used as a generic mapping between Button\+Handles. It maps an underlying \textquotesingle{}raw\textquotesingle{} button to a \textquotesingle{}virtual\textquotesingle{} button, which may optionally be associated with an appropriate platform-\/specific name for the button.
