If node\+\_\+path refers to a \doxylink{classGeomNode}{Geom\+Node(or has one beneath it)} the texture, its size, and UV data will be extracted from that.

If node\+\_\+path references a \doxylink{classSequenceNode}{Sequence\+Node(or has one beneath it)} with multiple Geom\+Nodes beneath it, the size data will correspond only to the first \doxylink{classGeomNode}{Geom\+Node} found with a valid texture, while the texture and UV information will be stored for each individual node.

If size\+\_\+from\+\_\+texels is true, the particle size is based on the number of texels in the source image; otherwise, it is based on the size of the first polygon found in the node.
