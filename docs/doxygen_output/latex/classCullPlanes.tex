\doxysubsection{Detailed Description}
This represents the set of clip planes and/or occluders that are definitely in effect for the current node of the \doxylink{classCullTraverserData}{Cull\+Traverser\+Data}, as well as on all child nodes. Any clip planes and occluders in this list may be safely culled against.

This does not include the clip planes that are in effect now, but might later be turned off by a child node, since we can\textquotesingle{}t safely cull against such clip planes.

The bounding volumes in this object are transformed for each level of the scene graph.
