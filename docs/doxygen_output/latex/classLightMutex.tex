\doxysubsection{Detailed Description}
This is a standard, non-\/reentrant mutex, similar to the \doxylink{classMutex}{Mutex} class. It is different from \doxylink{classMutex}{Mutex} in the case of SIMPLE\+\_\+\+THREADS\+: in this case, the \doxylink{classLightMutex}{Light\+Mutex} class compiles to nothing; it performs no locking whatsoever. It is therefore useful only to protect very small sections of code, during which you are confident there will be no thread yields.

In the normal, system-\/threaded implementation, this class is exactly the same as \doxylink{classMutex}{Mutex}.

Condition\+Vars cannot be used with \doxylink{classLightMutex}{Light\+Mutex}; they work only with \doxylink{classMutex}{Mutex}.

This class inherits its implementation either from Mutex\+Debug or \doxylink{classLightMutexDirect}{Light\+Mutex\+Direct}, depending on the definition of DEBUG\+\_\+\+THREADS.
