\doxysubsection{Detailed Description}
This class is an abstraction for evaluating NURBS surfaces. It accepts an array of vertices, each of which may be in a different coordinate space (as defined by a \doxylink{classNodePath}{Node\+Path}), as well as an optional knot vector.
