\doxysubsection{Detailed Description}
This class defines the physical layout of the vertex data stored within a \doxylink{classGeom}{Geom}. The layout consists of a list of named columns, each of which has a numeric type and a size.

The columns are typically interleaved within a single array, but they may also be distributed among multiple different arrays; at the extreme, each column may be alone within its own array (which amounts to a parallel-\/array definition).

Thus, a \doxylink{classGeomVertexFormat}{Geom\+Vertex\+Format} is really a list of Geom\+Vertex\+Array\+Formats, each of which contains a list of columns. However, a particular column name should not appear more than once in the format, even between different arrays.

There are a handful of standard pre-\/defined \doxylink{classGeomVertexFormat}{Geom\+Vertex\+Format} objects, or you may define your own as needed. You may record any combination of standard and/or user-\/defined columns in your custom \doxylink{classGeomVertexFormat}{Geom\+Vertex\+Format} constructions.
