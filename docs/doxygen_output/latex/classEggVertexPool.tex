\doxysubsection{Detailed Description}
A collection of vertices. There may be any number of vertex pools in a single egg structure. The vertices in a single pool need not necessarily have any connection to each other, but it is necessary that any one primitive (e.\+g. a polygon) must pull all its vertices from the same pool.

An \doxylink{classEggVertexPool}{Egg\+Vertex\+Pool} is an STL-\/style container of pointers to \doxylink{classEggVertex}{Egg\+Vertex}\textquotesingle{}s. Functions \doxylink{classEggVertexPool_ace396bc3d8610f4ae7bc8cc70d2da1ef}{add\+\_\+vertex()} and \doxylink{classEggVertexPool_a0d5176a3c1425ba5af9646c0116ade34}{remove\+\_\+vertex()} are provided to manipulate the list. The list may also be operated on (read-\/only) via iterators and \doxylink{classEggVertexPool_ab453b04185e440a165aaab166d9f71fc}{begin()}/end().
