\doxysubsection{Detailed Description}
A single declaration of a config variable, typically defined as one line in a .prc file, e.\+g. "{}show-\/frame-\/rate-\/meter 1"{}. This is really just a pairing of a string name (actually, a \doxylink{classConfigVariableCore}{Config\+Variable\+Core} pointer) to a string value.
