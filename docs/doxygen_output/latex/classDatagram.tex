\doxysubsection{Detailed Description}
An ordered list of data elements, formatted in memory for transmission over a socket or writing to a data file.

Data elements should be added one at a time, in order, to the \doxylink{classDatagram}{Datagram}. The nature and contents of the data elements are totally up to the user. When a \doxylink{classDatagram}{Datagram} has been transmitted and received, its data elements may be extracted using a \doxylink{classDatagramIterator}{Datagram\+Iterator}; it is up to the caller to know the correct type of each data element in order.

A \doxylink{classDatagram}{Datagram} is itself headerless; it is simply a collection of data elements.
