\doxysubsection{Detailed Description}
A \doxylink{classMovieAudio}{Movie\+Audio} is actually any source that provides a sequence of audio samples. That could include an AVI file, a microphone, or an internet TV station. A \doxylink{classMovieAudioCursor}{Movie\+Audio\+Cursor} is a handle that lets you read data sequentially from a \doxylink{classMovieAudio}{Movie\+Audio}.

\doxylink{classThread}{Thread} safety\+: each individual \doxylink{classMovieAudioCursor}{Movie\+Audio\+Cursor} must be owned and accessed by a single thread. It is OK for two different threads to open the same file at the same time, as long as they use separate \doxylink{classMovieAudioCursor}{Movie\+Audio\+Cursor} objects.
