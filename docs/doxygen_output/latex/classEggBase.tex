\doxysubsection{Detailed Description}
This is a base class for both \doxylink{classEggSingleBase}{Egg\+Single\+Base} and \doxylink{classEggMultiBase}{Egg\+Multi\+Base}. Don\textquotesingle{}t inherit directly from this; use one of those two classes instead.

This is just a base class; see \doxylink{classEggReader}{Egg\+Reader}, \doxylink{classEggWriter}{Egg\+Writer}, or \doxylink{classEggFilter}{Egg\+Filter} according to your particular I/O needs.
