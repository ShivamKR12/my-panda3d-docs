\doxysubsection{Detailed Description}
This represents a particular position of a texture within a \doxylink{classPaletteImage}{Palette\+Image}. There is only one of these per \doxylink{classTexturePlacement}{Texture\+Placement}, but it exists as a separate structure so the \doxylink{classTexturePlacement}{Texture\+Placement} can easily consider repositioning the texture.
