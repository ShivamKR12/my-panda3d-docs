{\footnotesize\ttfamily CPT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classGeomVertexData}{Geom\+Vertex\+Data}}}]{ }\end{DoxyParamCaption})}

Returns a \doxylink{classGeomVertexData}{Geom\+Vertex\+Data} that represents the results of computing the vertex animation on the CPU for this \doxylink{classGeom}{Geom}\textquotesingle{}s vertex data.

If there is no CPU-\/defined vertex animation on this object, this just returns the original object.

If there is vertex animation, but the \doxylink{classVertexTransform}{Vertex\+Transform} values have not changed since last time, this may return the same pointer it returned previously. Even if the \doxylink{classVertexTransform}{Vertex\+Transform} values have changed, it may still return the same pointer, but with its contents modified (this is preferred, since it allows the graphics backend to update vertex buffers optimally).

If force is false, this method may return immediately with stale data, if the vertex data is not completely resident. If force is true, this method will never return stale data, but may block until the data is available. Replaces the ith \doxylink{classGeomPrimitive}{Geom\+Primitive} object stored within the \doxylink{classGeom}{Geom} with the new object.
