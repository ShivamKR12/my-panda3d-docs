\doxysubsection{Detailed Description}
This global object records Node\+Paths that are referenced by scene graph attribs, such as Clip\+Plane\+Attribs and Light\+Attribs.

Its primary purpose is to unify attribs that are loaded in from bam files. Attrib nodes are identified by name and type; when a bam file that contains references to some attrib nodes is loaded, those nodes are first looked up here in the \doxylink{classAttribNodeRegistry}{Attrib\+Node\+Registry}. If there is a match (by name and node type), the identified node is used instead of the node referenced within the bam file itself.
