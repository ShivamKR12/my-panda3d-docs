\doxysubsection{Detailed Description}
This object provides a high-\/level interface for quickly writing a sequence of numeric values from a vertex table.

This object can be used both to replace existing vertices in the table, or to extend the table with new vertices. The set\+\_\+data\texorpdfstring{$\ast$}{*}() family of methods can only be used to replace existing data; it is an error to allow these to run past the end of the data. The add\+\_\+data\texorpdfstring{$\ast$}{*}() family of methods, on the other hand, can be used to replace existing data or add new data; if you call set\+\_\+row() into the middle of existing data the add\+\_\+data\texorpdfstring{$\ast$}{*}() methods will behave like the corresponding set\+\_\+data\texorpdfstring{$\ast$}{*}(), but if they run past the end of existing data they will quietly add new vertices.

Like \doxylink{classGeomVertexReader}{Geom\+Vertex\+Reader}, the writer is particularly optimized for writing a single column of data values for a series of vertices, without changing columns between each number. Although you can also use one \doxylink{classGeomVertexWriter}{Geom\+Vertex\+Writer} to write across the columns if it is convenient, by calling set\+\_\+column() repeatedly at each vertex, it is faster to write down the columns, and to use a different \doxylink{classGeomVertexWriter}{Geom\+Vertex\+Writer} for each column.

Note that, like a \doxylink{classGeomVertexReader}{Geom\+Vertex\+Reader}, a \doxylink{classGeomVertexWriter}{Geom\+Vertex\+Writer} does not keep a reference count to the actual vertex data buffer. This means that it is important not to keep a \doxylink{classGeomVertexWriter}{Geom\+Vertex\+Writer} object around over a long period of time in which the data buffer is likely to be deallocated; it is intended for making a quick pass over the data in one session.

It also means that you should create any Geom\+Vertex\+Writers {\itshape before} creating Geom\+Vertex\+Readers on the same data, since the writer itself might cause the vertex buffer to be deallocated. Better yet, use a \doxylink{classGeomVertexRewriter}{Geom\+Vertex\+Rewriter} if you are going to create both of them anyway.
