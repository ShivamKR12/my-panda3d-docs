\doxysubsection{Detailed Description}
An interface to the Tiny\+Panda software rendering code within this module.

Tiny\+Panda takes its name from Tiny\+GL, the public-\/domain software renderer (see \href{http://fabrice.bellard.free.fr/TinyGL/}{\texttt{ http\+://fabrice.\+bellard.\+free.\+fr/\+Tiny\+GL/}} ) from which this code originated. It has since been heavily modified, to integrate it closely with Panda, and to add additional features such as blending, filtering, and multitexturing.
