\doxysubsection{Detailed Description}
A special ostream that formats all of its output through Program\+Base\+::show\+\_\+text(). This allows the program to easily word-\/wrap its output messages to fit the terminal width.

By convention (inherited from show\+\_\+text), a newline written to the \doxylink{classWordWrapStream}{Word\+Wrap\+Stream} indicates a paragraph break, and is generally printed as a blank line. To force a line break without a paragraph break, use \textquotesingle{}\textbackslash{}r\textquotesingle{}.
