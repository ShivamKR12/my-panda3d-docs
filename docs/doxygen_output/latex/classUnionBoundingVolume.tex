\doxysubsection{Detailed Description}
This special bounding volume is the union of all of its constituent bounding volumes.

A point is defined to be within a \doxylink{classUnionBoundingVolume}{Union\+Bounding\+Volume} if it is within any one or more of its component bounding volumes.
