\doxysubsection{Detailed Description}
A class to manage a loose queue of isolated tasks, which can be performed either synchronously (in the foreground thread) or asynchronously (by a background thread).

The \doxylink{classAsyncTaskManager}{Async\+Task\+Manager} is actually a collection of Async\+Task\+Chains, each of which maintains a list of tasks. Each chain can be either foreground or background (it may run only in the main thread, or it may be serviced by one or more background threads). See \doxylink{classAsyncTaskChain}{Async\+Task\+Chain} for more information.

If you do not require background processing, it is perfectly acceptable to create only one \doxylink{classAsyncTaskChain}{Async\+Task\+Chain}, which runs in the main thread. This is a common configuration.
