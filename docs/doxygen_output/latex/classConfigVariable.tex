\doxysubsection{Detailed Description}
This is a generic, untyped \doxylink{classConfigVariable}{Config\+Variable}. It is also the base class for the typed Config\+Variables, and contains all of the code common to Config\+Variables of all types (except \doxylink{classConfigVariableList}{Config\+Variable\+List}, which is a bit of a special case).

Mostly, this class serves as a thin wrapper around \doxylink{classConfigVariableCore}{Config\+Variable\+Core} and/or \doxylink{classConfigDeclaration}{Config\+Declaration}, more or less duplicating the interface presented there.
