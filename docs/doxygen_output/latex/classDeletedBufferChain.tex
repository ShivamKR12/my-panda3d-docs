\doxysubsection{Detailed Description}
This template class can be used to provide faster allocation/deallocation for many Panda objects. It works by maintaining a linked list of deleted buffers that are all of the same size; when a new object is allocated that matches that size, the same space is just reused.

This class manages untyped buffers of a fixed size. It can be used directly; or it also serves as a backbone for \doxylink{classDeletedChain}{Deleted\+Chain}, which is a template class that manages object allocations.

Use \doxylink{classMemoryHook}{Memory\+Hook} to get a new \doxylink{classDeletedBufferChain}{Deleted\+Buffer\+Chain} of a particular size.
