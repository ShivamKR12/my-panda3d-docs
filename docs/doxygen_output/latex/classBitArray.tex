\doxysubsection{Detailed Description}
A dynamic array with an unlimited number of bits.

This is similar to a \doxylink{classBitMask}{Bit\+Mask}, except it appears to contain an infinite number of bits. You can use it very much as you would use a \doxylink{classBitMask}{Bit\+Mask}.
