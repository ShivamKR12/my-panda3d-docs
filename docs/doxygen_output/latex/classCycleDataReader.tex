\subsubsection*{template$<$class Cycle\+Data\+Type$>$\newline
class Cycle\+Data\+Reader$<$ Cycle\+Data\+Type $>$}
This template class calls Pipeline\+Cycler\+::read\+\_\+unlocked(), and then provides a transparent read-\/only access to the \doxylink{classCycleData}{Cycle\+Data}. It is used to access the data quickly, without holding a lock, for a thread that does not intend to modify the data and write it back out. For cases where the data might be subsequently modified, you should use \doxylink{classCycleDataLockedReader}{Cycle\+Data\+Locked\+Reader}.

It exists as a syntactic convenience to access the data in the \doxylink{classCycleData}{Cycle\+Data}. It also allows the whole system to compile down to nothing if DO\+\_\+\+PIPELINING is not defined.
