\doxysubsection{Detailed Description}
This flavor of \doxylink{classConnectionReader}{Connection\+Reader} will read from its sockets and queue up all of the datagrams read for later receipt by the client code. This class is useful for client code that doesn\textquotesingle{}t want to deal with threading and is willing to poll for datagrams at its convenience.
