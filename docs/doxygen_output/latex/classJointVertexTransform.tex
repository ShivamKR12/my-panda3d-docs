\doxysubsection{Detailed Description}
This is a specialization on \doxylink{classVertexTransform}{Vertex\+Transform} that returns the transform necessary to move vertices as if they were assigned to the indicated joint. The geometry itself should be parented to the scene graph at the level of the character\textquotesingle{}s root joint; that is, it should not be parented under a node directly animated by any joints.

Multiple combinations of these with different weights are used to implement soft-\/skinned vertices for an animated character.
