\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Const\+Pointer\+To$<$ T $>$}
A \doxylink{classConstPointerTo}{Const\+Pointer\+To} is similar to a \doxylink{classPointerTo}{Pointer\+To}, except it keeps a const pointer to the thing.

(Actually, it keeps a non-\/const pointer, because it must be allowed to adjust the reference counts, and it must be able to delete it when the reference count goes to zero. But it presents only a const pointer to the outside world.)

Notice that a \doxylink{classPointerTo}{Pointer\+To} may be assigned to a \doxylink{classConstPointerTo}{Const\+Pointer\+To}, but a \doxylink{classConstPointerTo}{Const\+Pointer\+To} may not be assigned to a \doxylink{classPointerTo}{Pointer\+To}.
