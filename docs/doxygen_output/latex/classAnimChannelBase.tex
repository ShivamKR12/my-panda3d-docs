\doxysubsection{Detailed Description}
\doxylink{classParent}{Parent} class for all animation channels. An \doxylink{classAnimChannel}{Anim\+Channel} is an arbitrary function that changes over time (actually, over frames), usually defined by a table read from an egg file (but possibly computed or generated in any other way).
