\doxysubsection{Detailed Description}
A specific kind of \doxylink{classDatagram}{Datagram}, especially for sending across or receiving from a network. It\textquotesingle{}s different only in that it knows which \doxylink{classConnection}{Connection} and/or \doxylink{classNetAddress}{Net\+Address} it is to be sent to or was received from.
