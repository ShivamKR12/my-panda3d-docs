\doxysubsection{Detailed Description}
A \doxylink{classStaticTextFont}{Static\+Text\+Font} is loaded up from a model that was previously generated via egg-\/mkfont, and contains all of its glyphs already generated and available for use. It doesn\textquotesingle{}t require linking with any external libraries like Free\+Type.
