\doxysubsection{Detailed Description}
An instance of this class is passed to the \doxylink{classFactory}{Factory} when requesting it to do its business and construct a new something. It can be filled with optional parameters to the Create\+Func for the particular subclass the \doxylink{classFactory}{Factory} will be creating.

This is just a vector of pointers to {\itshape something}; it will be up to the individual Create\+Funcs to interpret this meaningfully.
