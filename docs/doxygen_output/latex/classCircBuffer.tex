\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Thing, int max\+\_\+size$>$\newline
class Circ\+Buffer$<$ Thing, max\+\_\+size $>$}
This class implements a queue of some type via a circular buffer. The circular buffer has the advantage that no synchronization is required when one process adds to the queue while another process extracts. It works for any kind of Thing that has a valid assignment operator and copy constructor defined.
