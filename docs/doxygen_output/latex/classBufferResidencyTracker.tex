\doxysubsection{Detailed Description}
This class is used to keep track of the current state of all the Buffer\+Contexts for a particular graphics context\+: whether each one is active (rendered this frame) or inactive (not rendered this frame), and whether it is resident or nonresident in video memory.

The primary purpose of this class is to facilitate PStats reporting of video card memory usage.
