\doxysubsection{Detailed Description}
The specialization of \doxylink{classDXFLayer}{DXFLayer} used by \doxylink{classDXFToEggConverter}{DXFTo\+Egg\+Converter}. It contains a pointer to an \doxylink{classEggGroup}{Egg\+Group} and a vertex pool; these are used to build up polygons grouped by layer in the egg file as each polygon is read from the DXF file.
