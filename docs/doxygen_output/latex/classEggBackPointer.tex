\doxysubsection{Detailed Description}
This stores a pointer from an \doxylink{classEggJointData}{Egg\+Joint\+Data} or \doxylink{classEggSliderData}{Egg\+Slider\+Data} object back to the referencing data in an egg file. One of these objects corresponds to each model appearing in an egg file, and may reference either a single node, or a table, or a slew of vertices and primitives, depending on the type of data stored.

This is just an abstract base class. The actual details are stored in the various subclasses.
