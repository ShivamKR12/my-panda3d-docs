\doxysubsection{Detailed Description}
This is a single palette image, one of several within a \doxylink{classPalettePage}{Palette\+Page}, which is in turn one of several pages within a \doxylink{classPaletteGroup}{Palette\+Group}. Each palette image is a collage of several different textures that were all assigned to the same \doxylink{classPaletteGroup}{Palette\+Group}, and all share the same properties of the \doxylink{classPalettePage}{Palette\+Page}.
