\doxysubsection{Detailed Description}
Performs some generic munging that is appropriate for all GSG types; for instance, applies \doxylink{classColorAttrib}{Color\+Attrib} and \doxylink{classColorScaleAttrib}{Color\+Scale\+Attrib} to the vertices, and checks for hardware-\/accelerated animation capabilities.
