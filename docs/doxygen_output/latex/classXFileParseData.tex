\doxysubsection{Detailed Description}
This class is used to fill up the data into an \doxylink{classXFileDataNodeTemplate}{XFile\+Data\+Node\+Template} object as the data values are parsed out of the X file. It only has a temporary lifespan; it will be converted into actual data by \doxylink{classXFileDataNodeTemplate_a0b5aa3a2d089c65f8be8893fbd67b19f}{XFile\+Data\+Node\+Template\+::finalize\+\_\+parse\+\_\+data()}.
