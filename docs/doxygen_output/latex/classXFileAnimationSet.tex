\doxysubsection{Detailed Description}
This represents a tree of Egg\+Tables, corresponding to Animation entries in the X file. There is one \doxylink{classEggTable}{Egg\+Table} for each joint in the character\textquotesingle{}s joint set, and the whole tree is structured as a mirror of the joint set.
