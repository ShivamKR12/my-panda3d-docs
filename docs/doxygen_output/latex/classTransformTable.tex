\doxysubsection{Detailed Description}
Stores the total set of Vertex\+Transforms that the vertices in a particular \doxylink{classGeomVertexData}{Geom\+Vertex\+Data} object might depend on.

This structure is used for a \doxylink{classGeomVertexData}{Geom\+Vertex\+Data} set up to compute its dynamic vertices on the graphics card. See \doxylink{classTransformBlendTable}{Transform\+Blend\+Table} for one set up to compute its dynamic vertices on the CPU.
