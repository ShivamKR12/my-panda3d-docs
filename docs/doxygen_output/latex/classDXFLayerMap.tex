\doxysubsection{Detailed Description}
A map of string (layer name) to \doxylink{classDXFLayer}{DXFLayer}\+: that is, the layers of a file ordered by name. This is used as a lookup within \doxylink{classDXFFile}{DXFFile} to locate the layer associated with a particular entity.
