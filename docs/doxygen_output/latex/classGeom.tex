\doxysubsection{Detailed Description}
A container for geometry primitives. This class associates one or more \doxylink{classGeomPrimitive}{Geom\+Primitive} objects with a table of vertices defined by a \doxylink{classGeomVertexData}{Geom\+Vertex\+Data} object. All of the primitives stored in a particular \doxylink{classGeom}{Geom} are drawn from the same set of vertices (each primitive uses a subset of all of the vertices in the table), and all of them must be rendered at the same time, in the same graphics state.
