Constructs a new \doxylink{classCompassEffect}{Compass\+Effect} object. If the reference is an empty \doxylink{classNodePath}{Node\+Path}, it means the \doxylink{classCompassEffect}{Compass\+Effect} is relative to the root of the scene graph; otherwise, it\textquotesingle{}s relative to the indicated node. The properties bitmask specifies the set of properties that the compass node inherits from the reference instead of from its parent.
