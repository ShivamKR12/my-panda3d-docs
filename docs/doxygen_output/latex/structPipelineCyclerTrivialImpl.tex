\doxysubsection{Detailed Description}
This is the trivial, non-\/threaded implementation of Pipeline\+Cycler\+Base. It is only compiled when DO\+\_\+\+PIPELINING is not defined (which usually implies that threading is not available).

This implementation is designed to do as little as possible, and to compile to nothing, or almost nothing. It doesn\textquotesingle{}t actually support pipelining in any way. It doesn\textquotesingle{}t even perform any sanity checks to speak of. It\textquotesingle{}s designed for a strictly single-\/threaded application, and its purpose is to be as low-\/overhead as possible.

We define this as a struct instead of a class to emphasize the importance of byte placement within the object, so that the inherited struct\textquotesingle{}s data is likely to be placed by the compiler at the "{}this"{} pointer.
