\doxysubsection{Detailed Description}
A parametric curve defined by a sequence of control vertices, each with an in and out tangent.

This class is actually implemented as a \doxylink{classPiecewiseCurve}{Piecewise\+Curve} made up of several Cubic\+Curvesegs, each of which is created using the hermite\+\_\+basis() method. The \doxylink{classHermiteCurve}{Hermite\+Curve} class itself keeps its own list of the CV\textquotesingle{}s that are used to define the curve (since the \doxylink{classCubicCurveseg}{Cubic\+Curveseg} class doesn\textquotesingle{}t retain these).
