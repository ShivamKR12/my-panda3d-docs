\doxysubsection{Detailed Description}
Manages the communications to report statistics via a network connection to a remote \doxylink{classPStatServer}{PStat\+Server}.

Normally, there is only one \doxylink{classPStatClient}{PStat\+Client} in the world, although it is possible to have multiple PStat\+Clients if extraordinary circumstances require in. Since each \doxylink{classPStatCollector}{PStat\+Collector} registers itself with the \doxylink{classPStatClient}{PStat\+Client} when it is created, having multiple PStat\+Clients requires special care when constructing the various PStat\+Collectors.

If DO\+\_\+\+PSTATS is not defined, we don\textquotesingle{}t want to use stats at all. This class is therefore defined as a stub class.
