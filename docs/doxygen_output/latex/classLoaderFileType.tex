\doxysubsection{Detailed Description}
This is the base class for a family of scene-\/graph file types that the \doxylink{classLoader}{Loader} supports. Each kind of loader that\textquotesingle{}s available should define a corresponding \doxylink{classLoaderFileType}{Loader\+File\+Type} object and register itself.
