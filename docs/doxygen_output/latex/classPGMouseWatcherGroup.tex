\doxysubsection{Detailed Description}
This is a specialization on \doxylink{classMouseWatcherGroup}{Mouse\+Watcher\+Group}, to associate it with a \doxylink{classPGTop}{PGTop}. Originally we had \doxylink{classPGTop}{PGTop} multiply inheriting from Named\+Node and \doxylink{classMouseWatcherGroup}{Mouse\+Watcher\+Group}, but this causes problems with circular reference counts.
