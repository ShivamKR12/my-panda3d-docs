{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classCullBin}{Cull\+Bin}}}]{ }\end{DoxyParamCaption})}

Returns a newly-\/allocated \doxylink{classCullBin}{Cull\+Bin} object that contains a copy of just the subset of the data from this \doxylink{classCullBin}{Cull\+Bin} object that is worth keeping around for next frame.

If a particular \doxylink{classCullBin}{Cull\+Bin} object has no data worth preserving till next frame, it is acceptable to return NULL (which is the default behavior of this method). Called after all the geoms have been added, this indicates that the cull process is finished for this frame and gives the bins a chance to do any post-\/processing (like sorting) before moving on to draw. \Hypertarget{cullBin_8cxx_a9ac0fde6d037ac6d39a06e078216b8c9}\label{cullBin_8cxx_a9ac0fde6d037ac6d39a06e078216b8c9}

{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classPandaNode}{Panda\+Node}}}]{ }\end{DoxyParamCaption})}

Returns a special scene graph constructed to represent the results of the cull. This will be a single node with a list of \doxylink{classGeomNode}{Geom\+Node} children, which represent the various geom objects discovered by the cull.

This is useful mainly for high-\/level debugging and abstraction tools; it should not be mistaken for the low-\/level cull result itself. For the low-\/ level cull result, use draw() to efficiently draw the culled scene.
