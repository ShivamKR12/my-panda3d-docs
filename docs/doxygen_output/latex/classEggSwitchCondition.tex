\doxysubsection{Detailed Description}
This corresponds to a $<$\+Switch\+Condition$>$ entry within a group. It indicates the condition at which a level-\/of-\/detail is switched in or out. This is actually an abstract base class for potentially any number of specific different kinds of switching conditions; presently, only a $<$\+Distance$>$ type is actually supported.
