\doxysubsection{Detailed Description}
This is the base class of both \doxylink{classEggJointData}{Egg\+Joint\+Data} and \doxylink{classEggSliderData}{Egg\+Slider\+Data}. It represents a single component of a character, either a joint or a slider, along with back pointers to the references to this component in all model and animation egg files read.
