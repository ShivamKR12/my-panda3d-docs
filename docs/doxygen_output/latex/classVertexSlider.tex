\doxysubsection{Detailed Description}
This is an abstract base class that retains some slider value, which is a linear value that typically ranges from 0.\+0 to 1.\+0, and is used to control the animation of morphs (blend shapes).

It is similar to \doxylink{classVertexTransform}{Vertex\+Transform}, which keeps a full 4x4 transform matrix, but the \doxylink{classVertexSlider}{Vertex\+Slider} only keeps a single float value.
