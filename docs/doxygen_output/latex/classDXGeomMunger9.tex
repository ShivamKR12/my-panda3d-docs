\doxysubsection{Detailed Description}
This specialization on \doxylink{classGeomMunger}{Geom\+Munger} finesses vertices for DirectX rendering. In particular, it makes sure colors are stored in DirectX\textquotesingle{}s packed\+\_\+argb format, and that all relevant components are packed into a single array, in the correct order.
