\doxysubsection{Detailed Description}
This defines the actual numeric vertex data stored in a \doxylink{classGeom}{Geom}, in the structure defined by a particular \doxylink{classGeomVertexFormat}{Geom\+Vertex\+Format} object.

The data consists of one or more arrays, each of which in turn consists of a series of rows, one per vertex. All arrays should have the same number of rows; each vertex is defined by the column data from a particular row across all arrays.

Often, there will be only one array per \doxylink{classGeom}{Geom}, and the various columns defined in the \doxylink{classGeomVertexFormat}{Geom\+Vertex\+Format} will be interleaved within that array. However, it is also possible to have multiple different arrays, with a certain subset of the total columns defined in each array.

However the data is distributed, the effect is of a single table of vertices, where each vertex is represented by one row of the table.

In general, application code should not attempt to directly manipulate the vertex data through this structure; instead, use the \doxylink{classGeomVertexReader}{Geom\+Vertex\+Reader}, \doxylink{classGeomVertexWriter}{Geom\+Vertex\+Writer}, and \doxylink{classGeomVertexRewriter}{Geom\+Vertex\+Rewriter} objects to read and write vertex data at a high level.
