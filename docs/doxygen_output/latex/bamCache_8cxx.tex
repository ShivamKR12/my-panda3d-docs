{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classBamCacheRecord}{Bam\+Cache\+Record}}}]{ }\end{DoxyParamCaption})}

Looks up a file in the cache.

If the file is cacheable, then regardless of whether the file is found in the cache or not, this returns a \doxylink{classBamCacheRecord}{Bam\+Cache\+Record}. On the other hand, if the file cannot be cached, returns NULL.

If record-\/\texorpdfstring{$>$}{>}has\+\_\+data() returns true, then the file was found in the cache, and you may call record-\/\texorpdfstring{$>$}{>}extract\+\_\+data() to get the object. If record-\/\texorpdfstring{$>$}{>}has\+\_\+data() returns false, then the file was not found in the cache or the cache was stale; and you should reload the source file (calling record-\/\texorpdfstring{$>$}{>}add\+\_\+dependent\+\_\+file() for each file loaded, including the original source file), and then call record-\/\texorpdfstring{$>$}{>}set\+\_\+data() to record the resulting loaded object; and finally, you should call store() to write the cached record to disk.

Looks for the existing cache file that corresponds to the indicated filename. Normally, this is the specified cache filename exactly; but in the case of a hash collision, it may be a variant of the cache filename.

Reads the indicated cache file and returns its associated record if it can be read and it matches the source filename.

Actually reads a record from the file.
