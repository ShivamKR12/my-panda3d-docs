\doxysubsection{Detailed Description}
A special binner used only within this package to pre-\/process the egg tree for the loader and group things together as appropriate.

It is used to collect similar polygons together for a \doxylink{classGeom}{Geom}, as well as to group related LOD children together under a single LOD node.
