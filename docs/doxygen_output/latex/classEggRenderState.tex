\doxysubsection{Detailed Description}
This class is used within this package only to record the render state that should be assigned to each primitive. It is assigned to \doxylink{classEggPrimitive}{Egg\+Primitive} objects via the \doxylink{classEggBinner}{Egg\+Binner}.
