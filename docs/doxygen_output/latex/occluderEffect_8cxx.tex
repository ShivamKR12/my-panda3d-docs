{\footnotesize\ttfamily CPT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classRenderEffect}{Render\+Effect}}}]{ }\end{DoxyParamCaption})}

Constructs a new \doxylink{classOccluderEffect}{Occluder\+Effect} object that does nothing.

Returns a new \doxylink{classOccluderEffect}{Occluder\+Effect}, just like this one, but with the indicated occluder added to the list of occluders enabled by this effect. Returns a new \doxylink{classOccluderEffect}{Occluder\+Effect}, just like this one, but with the indicated occluder removed from the list of occluders enabled by this effect. Intended to be overridden by derived \doxylink{classOccluderEffect}{Occluder\+Effect} types to return a unique number indicating whether this \doxylink{classOccluderEffect}{Occluder\+Effect} is equivalent to the other one.

This should return 0 if the two \doxylink{classOccluderEffect}{Occluder\+Effect} objects are equivalent, a number less than zero if this one should be sorted before the other one, and a number greater than zero otherwise.

This will only be called with two \doxylink{classOccluderEffect}{Occluder\+Effect} objects whose get\+\_\+type() functions return the same. Tells the \doxylink{classBamReader}{Bam\+Reader} how to create objects of type \doxylink{classOccluderEffect}{Occluder\+Effect}.
