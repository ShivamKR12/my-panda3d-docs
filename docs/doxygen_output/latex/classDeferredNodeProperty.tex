\doxysubsection{Detailed Description}
This class keeps track of all the state we must make note of during the graph traversal, but cannot apply immediately. An instance of this class may be assigned to nodes as they are created, and then later, after the geometry has been created, the graph will be traversed again and the state will be applied.

This class is only local to this package; it is not exported.
