\doxysubsection{Detailed Description}
The overall manager of the network connections. This class gets the ball rolling; to use this package, you need to derive from this and define make\+\_\+monitor() to allocate and return a \doxylink{classPStatMonitor}{PStat\+Monitor} of the suitable type.

Then create just one \doxylink{classPStatServer}{PStat\+Server} object and call \doxylink{classPStatServer_ad3f87babd63a32b193b96f90c3cad21e}{listen()} with the port(s) you would like to listen on. It will automatically create PStat\+Monitors as connections are established and mark the connections closed as they are lost.
