A convenience function; the primary interface to this package. Loads up the indicated egg file, and returns the root of a scene graph. Returns NULL if the file cannot be read for some reason.

Also see the \doxylink{classEggLoader}{Egg\+Loader} class, which can exercise a bit more manual control over the loading process.
