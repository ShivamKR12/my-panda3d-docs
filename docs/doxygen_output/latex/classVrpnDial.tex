\doxysubsection{Detailed Description}
This is the actual interface to a particular VRPN dial device, and all of its numbered dials. A pointer to this object is stored in the \doxylink{classVrpnClient}{Vrpn\+Client} class for each differently-\/named VRPN dial device we connect to.

The VRPN callbacks go here, which in turn get vectored out to any \doxylink{classVrpnDialDevice}{Vrpn\+Dial\+Device} objects that register with this. When the last \doxylink{classVrpnDialDevice}{Vrpn\+Dial\+Device} object unregisters, the \doxylink{classVrpnDial}{Vrpn\+Dial} will be deleted by the \doxylink{classVrpnClient}{Vrpn\+Client}.

This class does not need to be exported from the DLL.
