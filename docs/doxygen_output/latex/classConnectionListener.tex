\doxysubsection{Detailed Description}
This is a special kind of \doxylink{classConnectionReader}{Connection\+Reader} that waits for activity on a rendezvous port and accepts a TCP connection (instead of attempting to read a datagram from the rendezvous port).

It is itself an abstract class, as it doesn\textquotesingle{}t define what to do with the established connection. See \doxylink{classQueuedConnectionListener}{Queued\+Connection\+Listener}.
