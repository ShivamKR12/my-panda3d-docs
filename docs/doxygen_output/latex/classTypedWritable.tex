Receives an array of pointers, one for each time manager-\/\texorpdfstring{$>$}{>}read\+\_\+pointer() was called in \doxylink{classTypedWritable_a3ad6cc1cd23bc3783d83edb3302bb43c}{fillin()}. Returns the number of pointers processed.

This is the callback function that is made by the \doxylink{classBamReader}{Bam\+Reader} at some later point, after all of the required pointers have been filled in. It is necessary because there might be forward references in a bam file; when we call read\+\_\+pointer() in \doxylink{classTypedWritable_a3ad6cc1cd23bc3783d83edb3302bb43c}{fillin()}, the object may not have been read from the file yet, so we do not have a pointer available at that time. Thus, instead of returning a pointer, read\+\_\+pointer() simply reserves a later callback. This function provides that callback. The calling object is responsible for keeping track of the number of times it called read\+\_\+pointer() and extracting the same number of pointers out of the supplied vector, and storing them appropriately within the object.

Reads the bytes created by a previous call to encode\+\_\+to\+\_\+bam\+\_\+stream(), and extracts the single object on those bytes. Returns true on success, false on error.

This variant sets the \doxylink{classTypedWritable}{Typed\+Writable} and \doxylink{classReferenceCount}{Reference\+Count} pointers separately; both are pointers to the same object. The reference count is not incremented; it is the caller\textquotesingle{}s responsibility to manage the reference count.

Note that this method cannot be used to retrieve objects that do not inherit from \doxylink{classReferenceCount}{Reference\+Count}, because these objects cannot persist beyond the lifetime of the \doxylink{classBamReader}{Bam\+Reader} that reads them. To retrieve these objects from a bam stream, you must construct a \doxylink{classBamReader}{Bam\+Reader} directly.

If you happen to know that the particular object in question inherits from \doxylink{classTypedWritableReferenceCount}{Typed\+Writable\+Reference\+Count} or \doxylink{classPandaNode}{Panda\+Node}, consider calling the variant of decode\+\_\+from\+\_\+bam\+\_\+stream() defined for those methods, which presents a simpler interface. \Hypertarget{classTypedWritable_a6d5ae5f57882895a98e7b3d6bb9f2d00}\label{classTypedWritable_a6d5ae5f57882895a98e7b3d6bb9f2d00}
