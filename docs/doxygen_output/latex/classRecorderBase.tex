\doxysubsection{Detailed Description}
This is the base class to a number of objects that record particular kinds of user input (like a \doxylink{classMouseRecorder}{Mouse\+Recorder}) to use in conjunction with a \doxylink{classRecorderController}{Recorder\+Controller} to record the user\textquotesingle{}s inputs for a session.

Note that \doxylink{classRecorderBase}{Recorder\+Base} does not actually inherit from \doxylink{classTypedObject}{Typed\+Object}, even though it defines get\+\_\+type(). The assumption is that the classes that derive from \doxylink{classRecorderBase}{Recorder\+Base} might also inherit independently from \doxylink{classTypedObject}{Typed\+Object}.

It also does not inherit from \doxylink{classTypedWritable}{Typed\+Writable}, but it defines a method called \doxylink{classRecorderBase_a719a7a79a0f7a11a9a4bd162095ff937}{write\+\_\+recorder()} which is very similar to a \doxylink{classTypedWritable}{Typed\+Writable}\textquotesingle{}s write\+\_\+datagram(). Classes that derive from \doxylink{classRecorderBase}{Recorder\+Base} and also inherit from \doxylink{classTypedWritable}{Typed\+Writable} may choose to remap \doxylink{classRecorderBase_a719a7a79a0f7a11a9a4bd162095ff937}{write\+\_\+recorder()} to do exactly the same thing as write\+\_\+datagram(), or they may choose to write something slightly different.

Most types of recorders should derive from Recorder, as it derives from \doxylink{classReferenceCount}{Reference\+Count}, except for \doxylink{classMouseRecorder}{Mouse\+Recorder}, which would otherwise doubly inherit from \doxylink{classReferenceCount}{Reference\+Count}.
