\doxysubsection{Detailed Description}
This is a particular collection of textures, within a \doxylink{classPaletteGroup}{Palette\+Group}, that all share the same \doxylink{classTextureProperties}{Texture\+Properties}. The textures on the same page may therefore all be placed on the same set of Palette\+Images together.
