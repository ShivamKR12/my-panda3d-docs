\doxysubsection{Detailed Description}
An object to create Graphics\+Outputs that share a particular 3-\/D API. Normally, there will only be one \doxylink{classGraphicsPipe}{Graphics\+Pipe} in an application, although it is possible to have multiple of these at once if there are multiple different API\textquotesingle{}s available in the same machine.

Often, the \doxylink{classGraphicsPipe}{Graphics\+Pipe} corresponds to a physical output device, hence the term "{}pipe"{}, but this is not necessarily the case.

The \doxylink{classGraphicsPipe}{Graphics\+Pipe} is used by the \doxylink{classGraphicsEngine}{Graphics\+Engine} object to create and destroy windows; it keeps ownership of the windows it creates.

M. Asad added new/interim functionality where \doxylink{classGraphicsPipe}{Graphics\+Pipe} now contains a device interface to directx/opengl which will be used to handle multiple windows from same device.
