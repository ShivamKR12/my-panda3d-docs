\doxysubsection{Detailed Description}
A class object that manages a single asynchronous audio load request. This works in conjunction with the \doxylink{classLoader}{Loader} class defined in pgraph, or really with any \doxylink{classAsyncTaskManager}{Async\+Task\+Manager}. Create a new \doxylink{classAudioLoadRequest}{Audio\+Load\+Request}, and add it to the loader via load\+\_\+async(), to begin an asynchronous load.
