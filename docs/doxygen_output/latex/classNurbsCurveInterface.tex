\doxysubsection{Detailed Description}
This abstract class defines the interface only for a Nurbs-\/style curve, with knots and coordinates in homogeneous space.

The \doxylink{classNurbsCurve}{Nurbs\+Curve} class inherits both from this and from \doxylink{classParametricCurve}{Parametric\+Curve}.
