This is called by \doxylink{classwglGraphicsWindow}{wgl\+Graphics\+Window} when it finds it cannot use the pfnum determined by the GSG. Assuming this pfnum corresponds to an "{}advanced"{} frame buffer determined by wgl\+Choose\+Pixel\+Format\+ARB, this asks the GSG to swap out that pfnum for the earlier, "{}preliminary"{} pfnum determined via Describe\+Pixel\+Format().

This is a one-\/way operation. Once called, you can never go back to the advanced pfnum.
