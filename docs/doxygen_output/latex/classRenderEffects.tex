\doxysubsection{Detailed Description}
This represents a unique collection of \doxylink{classRenderEffect}{Render\+Effect} objects that correspond to a particular renderable state.

You should not attempt to create or modify a \doxylink{classRenderEffects}{Render\+Effects} object directly. Instead, call one of the make() functions to create one for you. And instead of modifying a \doxylink{classRenderEffects}{Render\+Effects} object, create a new one.
