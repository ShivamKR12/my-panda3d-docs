This class is used by Egg\+Mesher\+::find\+\_\+fans() to attempt to make an \doxylink{classEggTriangleFan}{Egg\+Triangle\+Fan} out of the polygons connected to the indicated vertex.

Begins the fanning process. Searches for triangles and connects them into a fan.

In certain cases, if egg\+\_\+unroll\+\_\+fans is enabled, the resulting fan may be retesselated into a series of zig-\/zag triangles, which are stored in unrolled\+\_\+tris. Otherwise, an \doxylink{classEggMesherStrip}{Egg\+Mesher\+Strip} (representing the fan) is created and added to the mesher.
