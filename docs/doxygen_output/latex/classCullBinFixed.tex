\doxysubsection{Detailed Description}
A specific kind of \doxylink{classCullBin}{Cull\+Bin} that sorts geometry in the order specified by the user-\/specified draw\+\_\+order parameter. This allows precise relative ordering of two objects.

When two or more objects are assigned the same draw\+\_\+order, they are drawn in scene-\/graph order (as with \doxylink{classCullBinUnsorted}{Cull\+Bin\+Unsorted}).
