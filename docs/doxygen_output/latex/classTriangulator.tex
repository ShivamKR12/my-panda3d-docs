\doxysubsection{Detailed Description}
This class can triangulate a convex or concave polygon, even one with holes. It is adapted from an algorithm published as\+:

Narkhede A. and Manocha D., Fast polygon triangulation algorithm based on Seidel\textquotesingle{}s Algorithm, UNC-\/\+CH, 1994.

\href{http://www.cs.unc.edu/~dm/CODE/GEM/chapter.html}{\texttt{ http\+://www.\+cs.\+unc.\+edu/\texorpdfstring{$\sim$}{\string~}dm/\+CODE/\+GEM/chapter.\+html}}

It works strictly on 2-\/d points. See \doxylink{classTriangulator3}{Triangulator3} for 3-\/d points.
