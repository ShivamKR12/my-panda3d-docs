\subsubsection*{template$<$class BMType$>$\newline
class Double\+Bit\+Mask$<$ BMType $>$}
This is a special \doxylink{classBitMask}{Bit\+Mask} type that is implemented as a pair of lesser \doxylink{classBitMask}{Bit\+Mask} types, to present a double-\/wide bit mask. For instance, on a 32-\/bit system, this can be used to make a single 64-\/bit bit mask. More of these can be ganged up together to make a 128-\/bit mask, and so on.
