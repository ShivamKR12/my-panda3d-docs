{\footnotesize\ttfamily CPT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classGeomVertexFormat}{Geom\+Vertex\+Format}}}]{ }\end{DoxyParamCaption})}

Given a source \doxylink{classGeomVertexFormat}{Geom\+Vertex\+Format}, converts it if necessary to the appropriate format for rendering.

The protected implementation of premunge\+\_\+format(). This exists just to cast away the const pointer.

Adds the indicated format to the registry, if there is not an equivalent format already there; in either case, returns the pointer to the equivalent format now in the registry.

This must be called before a format may be used in a \doxylink{classGeom}{Geom}. After this call, you should discard the original pointer you passed in (which may or may not now be invalid) and let its reference count decrement normally; you should use only the returned value from this point on. \Hypertarget{bulletHelper_8cxx_af3117bcbed470f58cc5e989db146418c}\label{bulletHelper_8cxx_af3117bcbed470f58cc5e989db146418c}
