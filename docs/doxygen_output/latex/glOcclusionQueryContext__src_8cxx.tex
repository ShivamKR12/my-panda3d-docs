{\footnotesize\ttfamily bool CLP (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classOcclusionQueryContext}{Occlusion\+Query\+Context}}}]{ }\end{DoxyParamCaption})}

Returns true if the query\textquotesingle{}s answer is ready, false otherwise. If this returns false, the application must continue to poll until it returns true.

It is only valid to call this from the draw thread. Requests the graphics engine to expedite the pending answer--the application is now waiting until the answer is ready.
