\doxysubsection{Detailed Description}
This is a class designed to support low-\/overhead traversals of the complete scene graph, with a memory of the complete path through the graph at any given point.

You could just use a regular \doxylink{classNodePath}{Node\+Path} to do this, but since the \doxylink{classNodePath}{Node\+Path} requires storing Node\+Path\+Components on each node as it is constructed, and then removing them when it destructs, there is considerable overhead in that approach.

The \doxylink{classWorkingNodePath}{Working\+Node\+Path} eliminates this overhead (but does not guarantee consistency if the scene graph changes while the path is held).

At any given point, you may ask the \doxylink{classWorkingNodePath}{Working\+Node\+Path} for its actual \doxylink{classNodePath}{Node\+Path}, and it will construct and return a new \doxylink{classNodePath}{Node\+Path} representing the complete generated chain.
