\doxysubsection{Detailed Description}
This is a trivial class returned by Part\+Bundle\+Node\+::get\+\_\+bundle(). Its purpose is to hold the actual \doxylink{classPartBundle}{Part\+Bundle} pointer contained within the \doxylink{classPartBundleNode}{Part\+Bundle\+Node}, so that scene graph flatten operations can safely combine or duplicate Part\+Bundles as necessary without affecting high-\/level bundle operations.

The high-\/level Actor class defined in direct/src/actor, for instance, will store a list of Part\+Bundle\+Handles instead of on actual Part\+Bundles, so that it will be immune to changes from these flatten operations.
