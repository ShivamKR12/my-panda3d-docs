\doxysubsection{Detailed Description}
High level class for color interpolation. Segments must be added to the manager in order to achieve results using the "{}add\+\_\+\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}()"{} functions. Access to these segments is provided but not necessary general use.
