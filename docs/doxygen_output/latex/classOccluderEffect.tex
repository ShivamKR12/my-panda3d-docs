\doxysubsection{Detailed Description}
This functions similarly to a \doxylink{classLightAttrib}{Light\+Attrib} or \doxylink{classClipPlaneAttrib}{Clip\+Plane\+Attrib}. It indicates the set of occluders that modify the geometry at this level and below. Unlike a \doxylink{classClipPlaneAttrib}{Clip\+Plane\+Attrib}, an \doxylink{classOccluderEffect}{Occluder\+Effect} takes effect immediately when it is encountered during traversal, and thus can only add occluders; it may not remove them.
