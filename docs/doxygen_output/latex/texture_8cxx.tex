{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classAsyncFuture}{Async\+Future}}}]{ }\end{DoxyParamCaption})}

Schedules a background task that reloads the the \doxylink{classTexture}{Texture} from its disk file if there is not currently a RAM image (or uncompressed RAM image, if allow\+\_\+compression is false).

A higher priority value indicates that this texture should be reloaded sooner than textures with a lower priority value. If the reload hasn\textquotesingle{}t taken place yet, you can call this again to update the priority value.

If someone else reloads the texture using an explicit call to reload() while an async reload request is pending, the async reload request is cancelled.

Indicates that the texture should be enqueued to be prepared in the indicated prepared\+\_\+objects at the beginning of the next frame. This will ensure the texture is already loaded into texture memory if it is expected to be rendered soon.

Use this function instead of prepare\+\_\+now() to preload textures from a user interface standpoint. \Hypertarget{texture_8cxx_a6c1041453d1f783ea1f47dad0fc0d8ae}\label{texture_8cxx_a6c1041453d1f783ea1f47dad0fc0d8ae}

{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classTexturePeeker}{Texture\+Peeker}}}]{ }\end{DoxyParamCaption})}

Returns a \doxylink{classTexturePeeker}{Texture\+Peeker} object that can be used to examine the individual texels stored within this \doxylink{classTexture}{Texture} by (u, v) coordinate.

If the texture has a ram image resident, that image is used. If it does not have a full ram image but does have a simple\+\_\+ram\+\_\+image resident, that image is used instead. If neither image is resident the full image is reloaded.

Returns NULL if the texture cannot find an image to load, or the texture format is incompatible.
