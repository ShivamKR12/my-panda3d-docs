\doxysubsection{Detailed Description}
This represents a single source texture that is referenced by one or more egg files. It may be assigned to multiple \doxylink{classPaletteGroups}{Palette\+Groups}, and thus placed on multiple Palette\+Images (up to one per \doxylink{classPaletteGroup}{Palette\+Group}).

Since a \doxylink{classTextureImage}{Texture\+Image} may be referenced by multiple egg files that are each assigned to a different set of groups, it tries to maximize sharing between egg files and minimize the number of different \doxylink{classPaletteGroups}{Palette\+Groups} it is assigned to.
