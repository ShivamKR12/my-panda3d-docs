\doxysubsection{Detailed Description}
A class to retrieve the individual data elements previously stored in a \doxylink{classDatagram}{Datagram}. Elements may be retrieved one at a time; it is up to the caller to know the correct type and order of each element.

Note that it is the responsibility of the caller to ensure that the datagram object is not destructed while this \doxylink{classDatagramIterator}{Datagram\+Iterator} is in use.
