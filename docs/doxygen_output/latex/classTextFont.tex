\doxysubsection{Detailed Description}
An encapsulation of a font; i.\+e. a set of glyphs that may be assembled together by a \doxylink{classTextNode}{Text\+Node} to represent a string of text.

This is just an abstract interface; see \doxylink{classStaticTextFont}{Static\+Text\+Font} or Dynamic\+Text\+Font for an actual implementation.
