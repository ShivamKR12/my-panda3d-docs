\doxysubsection{Detailed Description}
A \doxylink{classRenderBuffer}{Render\+Buffer} is an arbitrary subset of the various layers (depth buffer, color buffer, etc.) of a drawing region. It consists of a \doxylink{classGraphicsStateGuardian}{Graphics\+State\+Guardian} pointer, along with a bitmask of the layers we\textquotesingle{}re interested in.
