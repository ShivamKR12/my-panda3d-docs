\doxysubsection{Detailed Description}
This class supervises the construction of an \doxylink{classEggData}{Egg\+Data} structure from the data represented by the \doxylink{classLwoHeader}{Lwo\+Header}. Reading and writing the egg and lwo structures is left to the user.
