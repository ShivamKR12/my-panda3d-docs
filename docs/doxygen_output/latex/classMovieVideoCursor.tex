\doxysubsection{Detailed Description}
A \doxylink{classMovieVideo}{Movie\+Video} is actually any source that provides a sequence of video frames. That could include an AVI file, a digital camera, or an internet TV station. A \doxylink{classMovieVideoCursor}{Movie\+Video\+Cursor} is a handle that lets you read data sequentially from a \doxylink{classMovieVideo}{Movie\+Video}.

\doxylink{classThread}{Thread} safety\+: each individual \doxylink{classMovieVideoCursor}{Movie\+Video\+Cursor} must be owned and accessed by a single thread. It is OK for two different threads to open the same file at the same time, as long as they use separate \doxylink{classMovieVideoCursor}{Movie\+Video\+Cursor} objects.
