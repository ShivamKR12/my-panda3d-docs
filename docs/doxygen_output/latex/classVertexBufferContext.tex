\doxysubsection{Detailed Description}
This is a special class object that holds all the information returned by a particular GSG to indicate the vertex data array\textquotesingle{}s internal context identifier.

This allows the GSG to cache the vertex data array in whatever way makes sense. For instance, DirectX can allocate a vertex buffer for the array. Open\+GL can create a buffer object.
