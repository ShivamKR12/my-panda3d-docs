\doxysubsection{Detailed Description}
The fundamental type of node for the data graph. The \doxylink{classDataNode}{Data\+Node} class is itself primarily intended as an abstract class; it defines no inputs and no outputs. Most kinds of data nodes will derive from this to specify the inputs and outputs in the constructor.

\doxylink{classDataNode}{Data\+Node} does not attempt to cycle its data with a \doxylink{structPipelineCycler}{Pipeline\+Cycler}. The data graph is intended to be used only within a single thread.
