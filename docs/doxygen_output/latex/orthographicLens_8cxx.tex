{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classLens}{Lens}}}]{ }\end{DoxyParamCaption})}

Allocates a new \doxylink{classLens}{Lens} just like this one. Returns true if the lens represents a linear projection (e.\+g. \doxylink{classPerspectiveLens}{Perspective\+Lens}, \doxylink{classOrthographicLens}{Orthographic\+Lens}), and therefore there is a valid matrix returned by get\+\_\+projection\+\_\+mat(), or false otherwise. Returns true if the lens represents a orthographic projection (i.\+e. it is a \doxylink{classOrthographicLens}{Orthographic\+Lens}), false otherwise. This is the generic implementation, which is based on do\+\_\+extrude() and assumes a linear distribution of depth values between the near and far points. Computes the complete transformation matrix from 3-\/d point to 2-\/d point, if the lens is linear.
