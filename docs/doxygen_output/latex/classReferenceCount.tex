\doxysubsection{Detailed Description}
A base class for all things that want to be reference-\/counted. \doxylink{classReferenceCount}{Reference\+Count} works in conjunction with \doxylink{classPointerTo}{Pointer\+To} to automatically delete objects when the last pointer to them goes away.
