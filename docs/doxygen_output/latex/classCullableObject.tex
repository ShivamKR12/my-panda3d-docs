\doxysubsection{Detailed Description}
The smallest atom of cull. This is normally just a \doxylink{classGeom}{Geom} and its associated state, but it also contain a draw callback.

Uses the indicated \doxylink{classGeomMunger}{Geom\+Munger} to transform the geom and/or its vertices.

If force is false, this may do nothing and return false if the vertex data is nonresident. If force is true, this will always return true, but it may have to block while the vertex data is paged in.
