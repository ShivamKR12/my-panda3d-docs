\doxysubsection{Detailed Description}
Defines the properties of a named stage of the multitexture pipeline. The \doxylink{classTextureAttrib}{Texture\+Attrib} will associated a number of these stages with \doxylink{classTexture}{Texture} objects, and the GSG will render geometry by sorting all of the currently active Texture\+Stages in order and then issuing the appropriate rendering calls to activate them.
