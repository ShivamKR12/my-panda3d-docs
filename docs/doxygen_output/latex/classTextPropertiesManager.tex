\doxysubsection{Detailed Description}
This defines all of the \doxylink{classTextProperties}{Text\+Properties} structures that might be referenced by name from an embedded text string.

A text string, as rendered by a \doxylink{classTextNode}{Text\+Node}, can contain embedded references to one of the \doxylink{classTextProperties}{Text\+Properties} defined here, by enclosing the name between \textbackslash{}1 (ASCII 0x01) characters; this causes a "{}push"{} to the named state. All text following the closing \textbackslash{}1 character will then be rendered in the new state. The next \textbackslash{}2 (ASCII 0x02) character will then restore the previous state for subsequent text.

For instance, "{}x\textbackslash{}1up\textbackslash{}1n\textbackslash{}2 + y"{} indicates that the character "{}x"{} will be rendered in the normal state, the character "{}n"{} will be rendered in the "{}up"{} state, and then "{} + y"{} will be rendered in the normal state again.

This can also be used to define arbitrary models that can serve as embedded graphic images in a text paragraph. This works similarly; the convention is to create a \doxylink{classTextGraphic}{Text\+Graphic} that describes the graphic image, and then associate it here via the \doxylink{classTextPropertiesManager_ad37cdf19996a4cbf32daebfcf3faedfa}{set\+\_\+graphic()} call. Then "{}\textbackslash{}5name\textbackslash{}5"{} will embed the named graphic.
