\doxysubsection{Detailed Description}
This is a base class for the various different classes that represent the result of a frame of rendering. The most common kind of \doxylink{classGraphicsOutput}{Graphics\+Output} is a \doxylink{classGraphicsWindow}{Graphics\+Window}, which is a real-\/time window on the desktop, but another example is \doxylink{classGraphicsBuffer}{Graphics\+Buffer}, which is an offscreen buffer.

The actual rendering, and anything associated with the graphics context itself, is managed by the associated \doxylink{classGraphicsStateGuardian}{Graphics\+State\+Guardian} (which might output to multiple \doxylink{classGraphicsOutput}{Graphics\+Output} objects).

Graphics\+Outputs are not actually writable to bam files, of course, but they may be passed as event parameters, so they inherit from \doxylink{classTypedWritableReferenceCount}{Typed\+Writable\+Reference\+Count} instead of \doxylink{classTypedReferenceCount}{Typed\+Reference\+Count} for that convenience.
