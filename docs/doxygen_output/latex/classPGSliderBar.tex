\doxysubsection{Detailed Description}
This is a particular kind of \doxylink{classPGItem}{PGItem} that draws a little bar with a slider that moves from left to right indicating a value between the ranges.

This is used as an implementation for both Direct\+Slider and for Direct\+Scroll\+Bar.
