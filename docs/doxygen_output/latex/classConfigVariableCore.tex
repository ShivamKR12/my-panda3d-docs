\doxysubsection{Detailed Description}
The internal definition of a \doxylink{classConfigVariable}{Config\+Variable}. This object is shared between all instances of a \doxylink{classConfigVariable}{Config\+Variable} that use the same variable name.

You cannot create a \doxylink{classConfigVariableCore}{Config\+Variable\+Core} instance directly; instead, use the make() method, which may return a shared instance. Once created, these objects are never destructed.
