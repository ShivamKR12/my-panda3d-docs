\doxysubsection{Detailed Description}
An abstract base class for gl\+Graphics\+Window and dx\+Graphics\+Window (and, in general, graphics windows that interface with the Microsoft Windows API).

This class includes all the code for manipulating windows themselves\+: opening them, closing them, responding to user keyboard and mouse input, and so on. It does not make any 3-\/D rendering calls into the window; that is the province of the \doxylink{classGraphicsStateGuardian}{Graphics\+State\+Guardian}.
