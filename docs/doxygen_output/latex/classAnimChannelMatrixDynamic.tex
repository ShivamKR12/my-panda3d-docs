\doxysubsection{Detailed Description}
An animation channel that accepts a matrix each frame from some dynamic input provided by code.

This object operates in two modes\+: in explicit mode, the programmer should call \doxylink{classAnimChannelMatrixDynamic_ad32d6079a4266bd807fdc4fa2115e48c}{set\+\_\+value()} each frame to indicate the new value; in implicit mode, the programmer should call \doxylink{classAnimChannelMatrixDynamic_ad7c385eb1412aa3487bf726bb5705ce3}{set\+\_\+value\+\_\+node()} to indicate the node whose transform will be copied to the joint each frame.
