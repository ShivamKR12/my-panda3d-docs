\doxysubsection{Detailed Description}
A specialized kind of \doxylink{classCollisionHandler}{Collision\+Handler} that sets the Z height of the collider to a fixed linear offset from the highest detected collision point each frame. It\textquotesingle{}s intended to implement walking around on a floor of varying height by casting a ray down from the avatar\textquotesingle{}s head.
