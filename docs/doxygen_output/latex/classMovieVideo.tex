\doxysubsection{Detailed Description}
A \doxylink{classMovieVideo}{Movie\+Video} is actually any source that provides a sequence of video frames. That could include an AVI file, a digital camera, or an internet TV station.

The difference between a \doxylink{classMovieVideo}{Movie\+Video} and a \doxylink{classMovieVideoCursor}{Movie\+Video\+Cursor} is like the difference between a filename and a file handle. The \doxylink{classMovieVideo}{Movie\+Video} just indicates a particular movie. The \doxylink{classMovieVideoCursor}{Movie\+Video\+Cursor} is what allows access.
