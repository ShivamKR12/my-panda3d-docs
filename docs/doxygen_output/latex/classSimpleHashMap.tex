\subsubsection*{template$<$class Key, class Value, class Compare = method\+\_\+hash$<$\+Key, std\+::less$<$\+Key$>$ $>$$>$\newline
class Simple\+Hash\+Map$<$ Key, Value, Compare $>$}
This template class implements an unordered map of keys to data, implemented as a hashtable. It is similar to STL\textquotesingle{}s hash\+\_\+map, but (a) it has a simpler interface (we don\textquotesingle{}t mess around with iterators), (b) it wants an additional method on the Compare object, Compare\+::is\+\_\+equal(a, b), (c) it doesn\textquotesingle{}t depend on the system STL providing hash\+\_\+map, (d) it allows for efficient iteration over the entries, (e) permits removal and resizing during forward iteration, and (f) it has a constexpr constructor.

It can also be used as a set, by using nullptr\+\_\+t as Value typename.
