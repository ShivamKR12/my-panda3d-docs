{\footnotesize\ttfamily void preprocess\+\_\+argv (\begin{DoxyParamCaption}\item[{int \&}]{argc,  }\item[{char \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\&}]{argv }\end{DoxyParamCaption})}

Processes the argc, argv pair as needed before passing it to getopt(). If this program is running on Windows, but not within Cygwin, this ignores the incoming argv, argv values, replacing them from the Get\+Command\+Line() string, and expanding glob patterns like \texorpdfstring{$\ast$}{*}.egg to a list of all matching egg files. On other platforms, this function does nothing and returns argc, argv unchanged.

The argc and argv values are modified by this function, if necessary, to point to statically-\/allocated memory that will be valid until the next call to \doxylink{preprocess__argv_8cxx_ac9ee2a106e3597173b4e370883cf9035}{preprocess\+\_\+argv()}.
