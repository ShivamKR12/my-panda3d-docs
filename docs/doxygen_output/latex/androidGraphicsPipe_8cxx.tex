Creates a new \doxylink{classGraphicsPipe}{Graphics\+Pipe} of the indicated type (or a type more specific than the indicated type, if necessary) and returns it. Returns NULL if the type cannot be matched.

Returns a new \doxylink{classGraphicsPipe}{Graphics\+Pipe} of a type defined by the indicated module. Returns NULL if the module is not found or does not properly recommend a \doxylink{classGraphicsPipe}{Graphics\+Pipe}.
