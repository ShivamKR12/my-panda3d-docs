\doxysubsection{Detailed Description}
This is a base class for those kinds of Saved\+Contexts that occupy an easily-\/measured (and substantial) number of bytes in the video card\textquotesingle{}s frame buffer memory or AGP memory. At the present, this includes most of the \doxylink{classSavedContext}{Saved\+Context} types\+: \doxylink{classVertexBufferContext}{Vertex\+Buffer\+Context} and \doxylink{classIndexBufferContext}{Index\+Buffer\+Context}, as well as \doxylink{classTextureContext}{Texture\+Context}.

This class provides methods for tracking the video memory utilization, as well as residency of each object, via PStats.
