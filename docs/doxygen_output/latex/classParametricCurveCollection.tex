\doxysubsection{Detailed Description}
This is a set of zero or more Parametric\+Curves, which may or may not be related. If they are related, the set should contain no more than one XYZ curve, no more than one HPR curve, and zero or more Timewarp curves, which can then be evaluated as a unit to return a single transformation matrix for a given unit of time.
