\doxysubsection{Detailed Description}
This specialization of \doxylink{classProgramBase}{Program\+Base} is intended for programs that read and/or write a single image file. (See Image\+Multi\+Base for programs that operate on multiple image files at once.)

This is just a base class; see \doxylink{classImageReader}{Image\+Reader}, \doxylink{classImageWriter}{Image\+Writer}, or \doxylink{classImageFilter}{Image\+Filter} according to your particular I/O needs.
