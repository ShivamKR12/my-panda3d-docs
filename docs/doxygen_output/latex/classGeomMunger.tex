\doxysubsection{Detailed Description}
Objects of this class are used to convert vertex data from a \doxylink{classGeom}{Geom} into a format suitable for passing to the rendering backend. Typically, the rendering backend will create a specialization of this class to handle its particular needs (e.\+g. DXGeom\+Munger). This class is necessary because DirectX and Open\+GL have somewhat different requirements for vertex format.

This also performs runtime application of state changes to the vertex data; for instance, by scaling all of the color values in response to a \doxylink{classColorScaleAttrib}{Color\+Scale\+Attrib}.

A \doxylink{classGeomMunger}{Geom\+Munger} must be registered before it can be used, and once registered, the object is constant and cannot be changed. All registered Geom\+Mungers that perform the same operation will have the same pointer.

Applies the indicated munger to the geom and its data, and returns a (possibly different) geom and data, according to the munger\textquotesingle{}s whim.

The assumption is that for a particular geom and a particular munger, the result will always be the same; so this result may be cached.

If force is false, this may do nothing and return false if the vertex data is nonresident. If force is true, this will always return true, but it may have to block while the vertex data is paged in. \Hypertarget{classGeomMunger_abfbe87f1c1d2e767b32bdc520413c0d0}\label{classGeomMunger_abfbe87f1c1d2e767b32bdc520413c0d0}
