\doxysubsection{Detailed Description}
A hierarchy of directories and files that appears to be one continuous file system, even though the files may originate from several different sources that may not be related to the actual OS\textquotesingle{}s file system.

For instance, a \doxylink{classVirtualFileSystem}{Virtual\+File\+System} can transparently mount one or more Multifiles as their own subdirectory hierarchies.
