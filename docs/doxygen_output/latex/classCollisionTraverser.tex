\doxysubsection{Detailed Description}
This class manages the traversal through the scene graph to detect collisions. It holds ownership of a number of collider objects, each of which is a \doxylink{classCollisionNode}{Collision\+Node} and an associated \doxylink{classCollisionHandler}{Collision\+Handler}.

When traverse() is called, it begins at the indicated root and detects all collisions with any of its collider objects against nodes at or below the indicated root, calling the appropriate \doxylink{classCollisionHandler}{Collision\+Handler} for each detected collision.
