\doxysubsection{Detailed Description}
This defines the abstract interface for an object that receives Geoms identified by the \doxylink{classCullTraverser}{Cull\+Traverser}. By itself, it\textquotesingle{}s not a particularly useful class; to use it, derive from it and redefine \doxylink{classCullHandler_a35e3215f1c647c0ce091d234a9afa57c}{record\+\_\+object()}.
