\subsubsection*{template$<$class T$>$\newline
class Weak\+Pointer\+To$<$ T $>$}
\doxylink{classWeakPointerTo}{Weak\+Pointer\+To} is similar to \doxylink{classPointerTo}{Pointer\+To}, except that it does not actually prevent the referenced pointer from deleting. Instead, the referenced pointer is allowed to delete, but if this happens then was\+\_\+deleted() will return true, and it will be an assertion error to dereference the pointer thereafter.
