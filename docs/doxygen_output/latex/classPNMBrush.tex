\doxysubsection{Detailed Description}
This class is used to control the shape and color of the drawing operations performed by a \doxylink{classPNMPainter}{PNMPainter} object.

Normally, you don\textquotesingle{}t create a \doxylink{classPNMBrush}{PNMBrush} directly; instead, use one of the static PNMBrush\+::make\+\_\+\texorpdfstring{$\ast$}{*}() methods provided here.

A \doxylink{classPNMBrush}{PNMBrush} is used to draw the border of a polygon or rectangle, as well as for filling its interior. When it is used to draw a border, the brush is "{}smeared"{} over the border; when it is used to fill the interior, it is tiled through the interior.
