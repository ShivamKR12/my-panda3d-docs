\doxysubsection{Detailed Description}
This is a special class object that holds all the information returned by a particular GSG to indicate the geom\textquotesingle{}s internal context identifier.

Geoms typically have an immediate-\/mode and a retained-\/mode operation. When using geoms in retained-\/mode (in response to \doxylink{classGeom_a0bf675037d2a13dfefa176d7c2be0486}{Geom\+::prepare()}), the GSG will create some internal handle for the geom and store it here. The geom stores all of these handles internally.

In the case of Open\+GL, for example, a \doxylink{classGeomContext}{Geom\+Context} corresponds to a display list identifier.
