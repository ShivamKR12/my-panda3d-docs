{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classLens}{Lens}}}]{ }\end{DoxyParamCaption})}

Allocates a new \doxylink{classLens}{Lens} just like this one. Returns true if the lens represents a linear projection (e.\+g. \doxylink{classPerspectiveLens}{Perspective\+Lens}, \doxylink{classMatrixLens}{Matrix\+Lens}), and therefore there is a valid matrix returned by get\+\_\+projection\+\_\+mat(), or false otherwise. Computes the complete transformation matrix from 3-\/d point to 2-\/d point, if the lens is linear.
