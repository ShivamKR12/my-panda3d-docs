\doxysubsection{Detailed Description}
This is a supporting class for \doxylink{classMemoryUsage}{Memory\+Usage}. It tracks the relative counts of a number of pointers of some type (or age), for use by Type\+Histogram and Age\+Histogram. It\textquotesingle{}s not exported from the DLL, and it doesn\textquotesingle{}t even exist if we\textquotesingle{}re compiling NDEBUG.
