\doxysubsection{Detailed Description}
This flavor of \doxylink{classConnectionManager}{Connection\+Manager} will queue up all of the reset-\/connection messages from the Connection\+Readers and Connection\+Writers and report them to the client on demand.

When a reset connection has been discovered via reset\+\_\+connection\+\_\+available()/get\+\_\+reset\+\_\+connection(), it is still the responsibility of the client to call close\+\_\+connection() on that connection to free up its resources.
