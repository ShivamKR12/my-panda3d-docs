\doxysubsection{Detailed Description}
Stored within \doxylink{classDXFFile}{DXFFile}, this is the basic Vertex data of a DXF file. When DXFFile\+::\+Done\+Entity() is called, if the entity is a type to have vertices, then DXFFile\+::\+\_\+verts contains a list of all the vertices that belong to the entity.
