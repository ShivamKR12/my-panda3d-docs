\doxysubsection{Detailed Description}
A special kind of task that serves as a list of tasks internally. Each task on the list is executed in sequence, one per epoch.

This is similar to a Sequence interval, though it has some slightly different abilities. For instance, although you can\textquotesingle{}t start at any arbitrary point in the sequence, you can construct a task sequence whose duration changes during playback.

Override this function to do something useful when the task has been removed from the active queue. The parameter clean\+\_\+exit is true if the task has been removed because it exited normally (returning DS\+\_\+done), or false if it was removed for some other reason (e.\+g. Async\+Task\+Manager\+::remove()). By the time this method is called, \+\_\+manager has been cleared, so the parameter manager indicates the original \doxylink{classAsyncTaskManager}{Async\+Task\+Manager} that owned this task.

The normal behavior is to throw the done\+\_\+event only if clean\+\_\+exit is true.

This function is called with the lock held. You may temporarily release if it necessary, but be sure to return with it held.
