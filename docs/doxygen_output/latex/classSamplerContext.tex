\doxysubsection{Detailed Description}
This is a special class object that holds a handle to the sampler state object given by the graphics back-\/end for a particular combination of texture sampling settings.

Some graphics back-\/ends (like Open\+GL) use mutable sampler objects, whereas others (Direct3D 10+) use immutable ones. In Panda3D, each unique sampler state has its own \doxylink{classSamplerContext}{Sampler\+Context}, which simplifies the implementation and makes redundant sampler objects impossible.
