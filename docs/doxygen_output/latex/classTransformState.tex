\doxysubsection{Detailed Description}
Indicates a coordinate-\/system transform on vertices. Transform\+States are the primary means for storing transformations on the scene graph.

Transforms may be specified in one of two ways\+: componentwise, with a pos-\/ hpr-\/scale, or with an arbitrary transform matrix. If you specify a transform componentwise, it will remember its original components.

\doxylink{classTransformState}{Transform\+State} objects are managed very much like \doxylink{classRenderState}{Render\+State} objects. They are immutable and reference-\/counted automatically.

You should not attempt to create or modify a \doxylink{classTransformState}{Transform\+State} object directly. Instead, call one of the make() functions to create one for you. And instead of modifying a \doxylink{classTransformState}{Transform\+State} object, create a new one.
