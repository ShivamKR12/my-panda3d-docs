\doxysubsection{Detailed Description}
An animation channel that accepts a scalar each frame from some dynamic input provided by code.

This object operates in two modes\+: in explicit mode, the programmer should call set\+\_\+value() each frame to indicate the new value; in implicit mode, the programmer should call \doxylink{classAnimChannelScalarDynamic_a70ebf7d50211510fed5d6bab3fd93b96}{set\+\_\+value\+\_\+node()} to indicate the node whose X component will be copied to the scalar each frame.
