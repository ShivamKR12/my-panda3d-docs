\doxysubsection{Detailed Description}
This is the base class of both \doxylink{classTextureImage}{Texture\+Image} and \doxylink{classPaletteImage}{Palette\+Image}. It encapsulates all the information specific to an image file that can be assigned as a texture image to egg geometry.
