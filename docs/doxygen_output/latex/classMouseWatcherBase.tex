\doxysubsection{Detailed Description}
This represents a collection of Mouse\+Watcher\+Regions that may be managed as a group. This is the base class for both \doxylink{classMouseWatcherGroup}{Mouse\+Watcher\+Group} and \doxylink{classMouseWatcher}{Mouse\+Watcher}, and exists so that we don\textquotesingle{}t have to make \doxylink{classMouseWatcher}{Mouse\+Watcher} inherit from \doxylink{classReferenceCount}{Reference\+Count} more than once.
