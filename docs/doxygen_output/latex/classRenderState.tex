\doxysubsection{Detailed Description}
This represents a unique collection of \doxylink{classRenderAttrib}{Render\+Attrib} objects that correspond to a particular renderable state.

You should not attempt to create or modify a \doxylink{classRenderState}{Render\+State} object directly. Instead, call one of the make() functions to create one for you. And instead of modifying a \doxylink{classRenderState}{Render\+State} object, create a new one.
