\doxysubsection{Detailed Description}
This is a special kind of frame that pretends to be much larger than it actually is. You can scroll through the frame, as if you\textquotesingle{}re looking through a window at the larger frame beneath. All children of this frame node are scrolled and clipped as if they were children of the larger, virtual frame.

This is implemented as a specialization of \doxylink{classPGVirtualFrame}{PGVirtual\+Frame}, which handles the meat of the virtual canvas. This class adds automatic support for scroll bars, and restricts the virtual transform to translate only (no scale or rotate).
