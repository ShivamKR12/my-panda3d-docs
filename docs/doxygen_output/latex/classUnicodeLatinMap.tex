\doxysubsection{Detailed Description}
This class mainly serves as a container for a largish table of the subset of the Unicode character set that corresponds to the Latin alphabet, with its various accent marks and so on. Specifically, this table indicates how to map between the Unicode accented character and the corresponding ASCII equivalent without the accent mark; as well as how to switch case from upper to lower while retaining the Unicode accent marks.
