\doxysubsection{Detailed Description}
This is a special \doxylink{classTextNode}{Text\+Node} that automatically updates itself with output from a \doxylink{classSceneGraphAnalyzer}{Scene\+Graph\+Analyzer} instance. It can be placed anywhere in the world where you\textquotesingle{}d like to see the output from \doxylink{classSceneGraphAnalyzer}{Scene\+Graph\+Analyzer}.

It also has a special mode in which it may be attached directly to a channel or window. If this is done, it creates a \doxylink{classDisplayRegion}{Display\+Region} for itself and renders itself in the upper-\/right-\/hand corner.
