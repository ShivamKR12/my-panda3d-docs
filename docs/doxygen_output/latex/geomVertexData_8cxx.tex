Returns a new \doxylink{classGeomVertexData}{Geom\+Vertex\+Data} object with the normal data modified in-\/place, so that each lighting normal is now facing in the opposite direction.

If the vertex data does not include a normal column, this returns the original \doxylink{classGeomVertexData}{Geom\+Vertex\+Data} object, unchanged. Returns a \doxylink{classGeomVertexData}{Geom\+Vertex\+Data} that represents the results of computing the vertex animation on the CPU for this \doxylink{classGeomVertexData}{Geom\+Vertex\+Data}.

If there is no CPU-\/defined vertex animation on this object, this just returns the original object.

If there is vertex animation, but the \doxylink{classVertexTransform}{Vertex\+Transform} values have not changed since last time, this may return the same pointer it returned previously. Even if the \doxylink{classVertexTransform}{Vertex\+Transform} values have changed, it may still return the same pointer, but with its contents modified (this is preferred, since it allows the graphics backend to update vertex buffers optimally).

If force is false, this method may return immediately with stale data, if the vertex data is not completely resident. If force is true, this method will never return stale data, but may block until the data is available. Removes the cache of animated vertices computed by a previous call to animate\+\_\+vertices() within the same frame.
