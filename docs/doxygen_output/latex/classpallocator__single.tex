\doxysubsection{Detailed Description}
\subsubsection*{template$<$class Type$>$\newline
class pallocator\+\_\+single$<$ Type $>$}
This is our own Panda specialization on the default STL allocator. Its main purpose is to call the hooks for \doxylink{classMemoryUsage}{Memory\+Usage} to properly track STL-\/ allocated memory.

pvector, pmap, etc. are all defined in this directory to use a pallocator.

pallocator actually comes it two flavors now\+: \doxylink{classpallocator__single}{pallocator\+\_\+single}, which can only allocate single instances of an object, and \doxylink{classpallocator__array}{pallocator\+\_\+array}, which can allocate arrays of objects.
