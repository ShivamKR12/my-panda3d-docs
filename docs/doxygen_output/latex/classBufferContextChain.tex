\doxysubsection{Detailed Description}
This class maintains a linked list of Buffer\+Contexts that might be allocated on the graphics card in some context. There is a different \doxylink{classBufferContextChain}{Buffer\+Context\+Chain} for resident textures, active textures, evicted textures, etc.

The primary purpose of this class is to facilitate PStats reporting of graphics memory usage.
