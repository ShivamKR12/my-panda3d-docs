\doxysubsection{Detailed Description}
This represents the in-\/memory index that records the list of files stored in the \doxylink{classBamCache}{Bam\+Cache}. Since the memory is also flushed to disk from time to time, this class is a \doxylink{classTypedWritable}{Typed\+Writable} object.

For the most part, this class is used only by the \doxylink{classBamCache}{Bam\+Cache} class.
