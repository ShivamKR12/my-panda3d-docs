\doxysubsection{Detailed Description}
This is the actual interface to a particular VRPN analog device, and all of its numbered controls. A pointer to this object is stored in the \doxylink{classVrpnClient}{Vrpn\+Client} class for each differently-\/named VRPN analog device we connect to.

The VRPN callbacks go here, which in turn get vectored out to any \doxylink{classVrpnAnalogDevice}{Vrpn\+Analog\+Device} objects that register with this. When the last \doxylink{classVrpnAnalogDevice}{Vrpn\+Analog\+Device} object unregisters, the \doxylink{classVrpnAnalog}{Vrpn\+Analog} will be deleted by the \doxylink{classVrpnClient}{Vrpn\+Client}.

This class does not need to be exported from the DLL.
