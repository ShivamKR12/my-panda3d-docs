Returns a full copy of the bundle and its entire tree of nested Anim\+Groups. However, the actual data stored in the leaves--that is, animation tables, such as those stored in an \doxylink{classAnimChannelMatrixXfmTable}{Anim\+Channel\+Matrix\+Xfm\+Table}--will be shared.
