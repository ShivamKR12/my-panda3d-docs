\doxysubsection{Detailed Description}
This class is the base class for both \doxylink{classConfigVariableList}{Config\+Variable\+List} and \doxylink{classConfigVariable}{Config\+Variable} (and hence for all of the \doxylink{classConfigVariableBool}{Config\+Variable\+Bool}, Config\+Varible\+String, etc. classes). It collects together the common interface for all generic Config\+Variables.

Mostly, this class serves as a thin wrapper around \doxylink{classConfigVariableCore}{Config\+Variable\+Core} and/or \doxylink{classConfigDeclaration}{Config\+Declaration}, more or less duplicating the interface presented there.
