\doxysubsection{Detailed Description}
\doxylink{classNodePath}{Node\+Path} is the fundamental system for disambiguating instances, and also provides a higher-\/level interface for manipulating the scene graph.

A \doxylink{classNodePath}{Node\+Path} is a list of connected nodes from the root of the graph to any sub-\/node. Each \doxylink{classNodePath}{Node\+Path} therefore uniquely describes one instance of a node.

Node\+Paths themselves are lightweight objects that may easily be copied and passed by value. Their data is stored as a series of Node\+Path\+Components that are stored on the nodes. Holding a \doxylink{classNodePath}{Node\+Path} will keep a reference count to all the nodes in the path. However, if any node in the path is removed or reparented (perhaps through a different \doxylink{classNodePath}{Node\+Path}), the \doxylink{classNodePath}{Node\+Path} will automatically be updated to reflect the changes.
