\doxysubsection{Detailed Description}
This is a special extension to \doxylink{classReferenceCount}{Reference\+Count} that includes dual reference counts\+: the standard reference count number, which includes all references to the object, and a separate number (the cache reference count) that counts the number of references to the object just within its cache alone. When get\+\_\+ref\+\_\+count() == get\+\_\+cache\+\_\+ref\+\_\+count(), the object is not referenced outside the cache.

The cache refs must be explicitly maintained; there is no Pointer\+To$<$$>$ class to maintain the cache reference counts automatically. The cache reference count is automatically included in the overall reference count\+: calling cache\+\_\+ref() and cache\+\_\+unref() automatically calls ref() and unref().
