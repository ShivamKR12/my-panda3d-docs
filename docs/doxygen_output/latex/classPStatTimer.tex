\doxysubsection{Detailed Description}
A lightweight class that can be used to automatically start and stop a \doxylink{classPStatCollector}{PStat\+Collector} around a section of code. It\textquotesingle{}s intended to be used in the following way\+: create a local \doxylink{classPStatTimer}{PStat\+Timer} variable to start the Collector, and when the \doxylink{classPStatTimer}{PStat\+Timer} variable goes out of scope (for instance, at the end of the function), it will automatically stop the Collector.
