\doxysubsection{Detailed Description}
The set of attributes that may be applied to vertices as well as polygons, such as surface normal and color.

This class cannot inherit from \doxylink{classEggObject}{Egg\+Object}, because it causes problems at the \doxylink{classEggPolygon}{Egg\+Polygon} level with multiple appearances of the \doxylink{classEggObject}{Egg\+Object} base class. And making \doxylink{classEggObject}{Egg\+Object} a virtual base class is just no fun.
