{\footnotesize\ttfamily PT (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classEggData}{Egg\+Data}}}]{ }\end{DoxyParamCaption})}

Allocates and returns a new \doxylink{classEggData}{Egg\+Data} structure that represents the indicated egg file. If the egg file cannot be read for some reason, returns NULL.

This can be overridden by derived classes to control how the egg files are read, or to extend the information stored with each egg structure, by deriving from \doxylink{classEggData}{Egg\+Data}.
