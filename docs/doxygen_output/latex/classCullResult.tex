\doxysubsection{Detailed Description}
This stores the result of a \doxylink{classBinCullHandler}{Bin\+Cull\+Handler} traversal\+: an ordered collection of Cull\+Bins, each of which holds a number of Geoms and Render\+States to be rendered in some defined order.

This is also used to keep the results of last frame\textquotesingle{}s cull traversal around to make next frame\textquotesingle{}s traversal of the same scene a little easier.
