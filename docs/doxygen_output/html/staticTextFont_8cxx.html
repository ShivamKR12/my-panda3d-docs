PT ( TextFont  )

Returns a new copy of the same font. Gets the glyph associated with the given character code, as well as an optional scaling parameter that should be applied to the glyph's geometry and advance parameters. Returns true if the glyph exists, false if it does not. Even if the return value is false, the value for glyph might be filled in with a printable glyph.
