Detailed Description

A single CV of a Hermite curve. Hermite curve CV's include an in and out tangent, as well as a position.

◆ set_type()

void HermiteCurveCV::set_type ( int  type)

Sets the continuity type of the CV. Values may be HC_CUT, indicating a discontinous break in the curve, HC_FREE, for unconstrained in and out tangents, HC_G1, for in and out tangents constrained to be collinear, and HC_SMOOTH, for in and out tangents constrained to be equal. Other than HC_CUT, these are for documentation only; setting this has no direct effect on the tangents.
