PT ( Texture  )

This method is called before each texture is loaded from disk, via the TexturePool, for the first time. If this method returns NULL, then a new Texture will be allocated and loaded from disk normally by the TexturePool; otherwise, if it returns non-NULL, then that returned pointer will be used as the Texture for this filename.

This method is called after each texture has been loaded from disk, via the TexturePool, for the first time. By the time this method is called, the Texture has already been fully read from disk. This method should return the Texture pointer that the TexturePool should actually return (usually it is the same as the pointer supplied).
