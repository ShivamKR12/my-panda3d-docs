14// These functions return a matrix that rotates between a coordinate system
15// defined with the given forward and up vectors, and the standard coordinate
16// system with y-forward and z-up. They differ only in their behavior when
17// the supplied forward and up vectors are not perpendicular; in this case,
18// look_at will match the forward vector precisely, while heads_up will match
19// the up vector precisely.
20
21// Since these functions only return a rotation matrix, the translation
22// component is always zero. There are flavors of these functions that simply
23// return the upper 3x3 part of the matrix, and flavors that return the whole
24// 4x4 matrix with a zero bottom row.
