Detailed Description

A collection of Geoms and their associated state, for a particular scene. The cull traversal (and the BinCullHandler) assigns Geoms to bins as it comes across them.

This is an abstract base class; derived classes like CullBinStateSorted and CullBinBackToFront provide the actual implementation.
