Detailed Description

A map of string (layer name) to DXFLayer: that is, the layers of a file ordered by name. This is used as a lookup within DXFFile to locate the layer associated with a particular entity.

DXFLayer * DXFLayerMap::get_layer ( const std::string &  name,
DXFFile *  dxffile 
)

Looks up the layer name in the map, and returns a pointer to the associated DXFLayer. If this is the first time this layer name has been used, creates a new DXFLayer by the given name. In this case, it calls dxffile->new_layer() to create the layer, allowing user code to override this function to create a specialized time, if desired.
