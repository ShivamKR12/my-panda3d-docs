Detailed Description

A reference to an EggNurbsSurface in the egg file, and its parameters as set by the user input file and as computed in relation to the other surfaces.

Member Function Documentation

◆ get_score()

double QtessSurface::get_score ( double  ratio)

Computes the curvature/stretch score for the surface, if it has not been already computed, and returns the net surface score. This is used both for automatically distributing isoparams among the surfaces by curvature, as well as for automatically placing the isoparams within each surface by curvature.
