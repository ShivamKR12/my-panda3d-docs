Detailed Description

Represents a single material in the material palette.

bool FltMaterial::build_14_record ( Datagram &  datagram)

Fills up the current record on the FltRecordWriter with data for this record, formatted as a part of a v14 material palette. Returns true on success, false if there is some error.
