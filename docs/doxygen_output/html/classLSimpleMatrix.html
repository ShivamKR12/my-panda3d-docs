Detailed Description

template<class FloatType, int NumRows, int NumCols>
class LSimpleMatrix< FloatType, NumRows, NumCols >

This class provides an underlying storage of the various linear-algebra classes (e.g. LVecBase3, LMatrix4) in the absence of the Eigen linear algebra library.
