Detailed Description

◆ is_template_def()

bool XFileTemplate::is_template_def ( ) const
virtual

Returns true if this node represents the definition of some template. This is the template definition, not an actual data object that represents an instance of the template. If the file strictly uses standard templates, the presence of template definitions is optional.

If this returns true, the node must be of type XFileTemplate.

Reimplemented from XFileNode.
