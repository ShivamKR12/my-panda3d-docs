Detailed Description

The ShaderContext is meant to contain the compiled version of a shader string. ShaderContext is an abstract base class, there will be a subclass of it for each shader language and graphics API. Since the languages are so different and the graphics APIs have so little in common, the base class contains almost nothing. All the implementation details are in the subclasses.
