PT ( Lens  )

Allocates a new Lens just like this one. Returns true if the lens represents a linear projection (e.g. PerspectiveLens, OrthographicLens), and therefore there is a valid matrix returned by get_projection_mat(), or false otherwise. Returns true if the lens represents a perspective projection (i.e. it is a PerspectiveLens), false otherwise. This is the generic implementation, which is based on do_extrude() and assumes a linear distribution of depth values between the near and far points. Computes the complete transformation matrix from 3-d point to 2-d point, if the lens is linear.
