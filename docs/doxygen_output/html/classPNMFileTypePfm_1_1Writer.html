bool PNMFileTypePfm::Writer::supports_floating_point ( )
virtual

Returns true if this PNMFileType can accept a floating-point image type, false if it can only accept a normal, integer type. If this returns true, write_pfm() is implemented.

Reimplemented from PNMWriter.

◆ supports_integer()

bool PNMFileTypePfm::Writer::supports_integer ( )
virtual

Returns true if this PNMFileType can accept an integer image type, false if it can only accept a floating-point type. If this returns true, write_data() or write_row() is implemented.

Reimplemented from PNMWriter.
