◆ get_current_pstat_major_version()

EXPCL_PANDA_PSTATCLIENT int get_current_pstat_major_version ( )

Returns the current major version number of the PStats protocol. This is the version number that will be reported by clients running this code, and that will be expected by servers running this code.

The major version numbers must match exactly in order for a communication to be successful.

◆ get_current_pstat_minor_version()

EXPCL_PANDA_PSTATCLIENT int get_current_pstat_minor_version ( )

Returns the current minor version number of the PStats protocol. This is the version number that will be reported by clients running this code, and that will be expected by servers running this code.

The minor version numbers need not match exactly, but the server must be >= the client.
