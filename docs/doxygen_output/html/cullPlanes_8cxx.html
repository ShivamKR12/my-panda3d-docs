CPT ( CullPlanes  )

Returns a pointer to an empty CullPlanes object.

Returns a pointer to a new CullPlanes object that is the same as this one, but with the clip planes modified by the indicated transform. Returns a pointer to a new CullPlanes object that is the same as this one, but with the indicated attributes applied to the state.

In particular, any new ClipPlanes given in net_attrib, if it is not NULL, will be added to the state, unless those ClipPlanes are also listed in off_attrib.
