Detailed Description

This is an abstract class that presents the interface for drawing a flame chart: it shows the time spent in each of a number of collectors as a horizontal bar of color, with time as the horizontal axis.

This class just manages all the flame chart logic; the actual nuts and bolts of drawing pixels is left to a user-derived class.
