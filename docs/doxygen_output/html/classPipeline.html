Detailed Description

This class manages a staged pipeline of data, for instance the render pipeline, so that each stage of the pipeline can simultaneously access different copies of the same data. It actually maintains a collection of PipelineCycler objects, and manages the turning of all of them at once.

There is one default Pipeline object, the render pipeline. Other specialty pipelines may be created as needed.
