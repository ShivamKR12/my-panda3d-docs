Detailed Description

Defines one level of array bounds for an associated XFileDataDef element.

Member Function Documentation

◆ get_size()

int XFileArrayDef::get_size ( const XFileNode::PrevData &  prev_data) const

Returns the size of the array dimension. If this is a fixed array, the size is trivial; if it is dynamic, the size is determined by looking up the dynamic_size element in the prev_data table (which lists all of the data values already defined at this scoping level).
