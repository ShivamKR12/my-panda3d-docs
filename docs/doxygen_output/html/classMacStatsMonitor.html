Detailed Description

This class represents a connection to a PStatsClient and manages the data exchange with the client.

◆ initialized()

void MacStatsMonitor::initialized ( )
virtual

Called after the monitor has been fully set up. At this time, it will have a valid _client_data pointer, and things like is_alive() and close() will be meaningful. However, we may not yet know who we're connected to (is_client_known() may return false), and we may not know anything about the threads or collectors we're about to get data on.

Reimplemented from PStatMonitor.

◆ new_collector()

void MacStatsMonitor::new_collector ( int  collector_index)
virtual

Called whenever a new Collector definition is received from the client. Generally, the client will send all of its collectors over shortly after connecting, but there's no guarantee that they will all be received before the first frames are received. The monitor should be prepared to accept new Collector definitions midstream.

Reimplemented from PStatMonitor.
