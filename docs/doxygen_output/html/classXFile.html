Detailed Description

This represents the complete contents of an X file (file.x) in memory. It may be read or written from or to a disk file.

◆ read()

bool XFile::read ( Filename  filename)

Opens and reads the indicated .x file by name. The nodes and templates defined in the file will be appended to the set of nodes already recorded, if any.

Returns true if the file is successfully read, false if there was an error (in which case the file might have been partially read).
