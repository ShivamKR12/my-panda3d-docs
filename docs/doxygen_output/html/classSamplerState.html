Detailed Description

Represents a set of settings that indicate how a texture is sampled. This can be used to sample the same texture using different settings in different places.

◆ compare_to()

int SamplerState::compare_to ( const SamplerState &  other) const

Returns a number less than zero if this sampler sorts before the other one, greater than zero if it sorts after, or zero if they are equivalent. The sorting order is arbitrary and largely meaningless, except to differentiate different sampler states.
