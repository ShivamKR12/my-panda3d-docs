◆ supports_convert_to_node()

bool ObjToEggConverter::supports_convert_to_node ( const LoaderOptions &  options) const
virtual

Returns true if this converter can directly convert the model type to internal Panda memory structures, given the indicated options, or false otherwise. If this returns true, then convert_to_node() may be called to perform the conversion, which may be faster than calling convert_file() if the ultimate goal is a PandaNode anyway.
