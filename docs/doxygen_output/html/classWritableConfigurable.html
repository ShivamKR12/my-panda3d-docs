Detailed Description

Defined as a fix to allow creating Configurable and Writable objects. Otherwise the compiler gets confused since both TypedWritable and Configurable inherit from TypedObject.

An object that has data or parameters that are set less frequently (at least occasionally) than every frame. We can cache the configuration info by by using the "dirty" flag.
