Detailed Description

This is an object shared by all the weak pointers that point to the same ReferenceCount object. It is created whenever a weak reference to an object is created, and can outlive the object until all weak references have disappeared.

void WeakReferenceList::add_callback ( WeakPointerCallback *  callback,
void *  data

Adds the callback to the list of callbacks that will be called when the underlying pointer is deleted. If it has already been deleted, it will be called immediately.

The data pointer can be an arbitrary pointer and is passed as only argument to the callback.
