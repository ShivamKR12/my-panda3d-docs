278 // A node has three bounding volumes: an "external" bounding volume that
279 // represents the node and all of its children, an "internal" bounding
280 // volume which represents only the node itself (and is usually empty,
281 // unless a specific node type sets it otherwise), and a "user" bounding
282 // volume which is specified by the user.
283
284 // We define set_bounds() and get_bounds() functions so that set_bounds()
285 // sets the user bounding volume, while get_bounds() returns the external
286 // bounding volume. Although it might seem strange and confusing to do
287 // this, this is actually the natural way the user thinks about nodes and
288 // bounding volumes.
