Detailed Description

A generic DXF-reading class. This class can read a DXF file but doesn't actually do anything with the data; it's intended to be inherited from and the appropriate functions overridden (particularly DoneEntity()).

◆ done_entity()

void DXFFile::done_entity ( )
virtual

This is the primary hook for user code. This function is called when an entity is read from the DXF file. This may be something like a polygon, point, or a polygon mesh: any geometry. It is up to the user code to override this function and do something interesting with each piece of geometry that is read.

Reimplemented in DXFToEggConverter, and DXFPoints.

◆ done_vertex()

void DXFFile::done_vertex ( )
virtual

A hook for user code, if desired. This function is called whenever a vertex is read from the DXF file. This function has the default behavior of adding the vertex to the _verts list, so that when done_entity() is called later, it will have the complete list of vertices available to it.

◆ ocs_2_wcs()

void DXFFile::ocs_2_wcs ( )

Assuming the current entity is a planar-based entity, for instance, a 2-d polygon (as opposed to a 3-d polygon), this converts the coordinates from the funny planar coordinate system to the world coordinates. It converts the _p value of the entity, as well as all vertices in the _verts list.
