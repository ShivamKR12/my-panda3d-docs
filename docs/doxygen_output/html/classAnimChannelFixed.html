Detailed Description

template<class SwitchType>
class AnimChannelFixed< SwitchType >

This template class is a special kind of AnimChannel that always returns just one fixed value. It is a special channel, in that it need not be assigned within a hierarchy. It may stand alone, so that it may be created on-the-fly for parts that need default anims to bind against.

Returns true if the value has changed since the last call to has_changed(). last_frame is the frame number of the last call; this_frame is the current frame number. last_frac and this_frac are the fractional steps into those frames, which will be 0.0 unless we are running in frame_blend mode.
