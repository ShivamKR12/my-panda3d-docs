Detailed Description

This class provides a locking wrapper around an arbitrary istream pointer. A thread may use this class to perform an atomic seek/read/gcount operation.

Member Function Documentation

◆ seek_gpos_eof()

streamsize IStreamWrapper::seek_gpos_eof ( )

Atomically seeks to EOF and returns the gpos there; that is, returns the file size. Note that the EOF might have been moved in another thread by the time this method returns.
