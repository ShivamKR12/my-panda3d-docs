Detailed Description

This graphics pipe represents the interface for creating OpenGL graphics windows on the various IPHONE's.

◆ get_interface_name()

std::string IPhoneGraphicsPipe::get_interface_name ( ) const
virtual

Returns the name of the rendering interface associated with this GraphicsPipe. This is used to present to the user to allow him/her to choose between several possible GraphicsPipes available on a particular platform, so the name should be meaningful and unique for a given platform.
