Detailed Description

High level class for color interpolation. Segments must be added to the manager in order to achieve results using the "add_*****()" functions. Access to these segments is provided but not necessary general use.

Member Function Documentation

◆ generateColor()

LColor ColorInterpolationManager::generateColor ( const PN_stdfloat  interpolated_time)

For time 'interpolated_time', this returns the additive composite color of all segments that influence that instant in the particle's lifetime. If no segments cover that time, the manager's default color is returned.
