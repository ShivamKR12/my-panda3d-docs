Detailed Description

A single declaration of a config variable, typically defined as one line in a .prc file, e.g. "show-frame-rate-meter 1". This is really just a pairing of a string name (actually, a ConfigVariableCore pointer) to a string value.

Divides the string into a number of words according to whitespace. The words vector should be cleared by the user before calling; otherwise, the list of words in the string will be appended to the end of whatever was there before.

The return value is the number of words extracted.
