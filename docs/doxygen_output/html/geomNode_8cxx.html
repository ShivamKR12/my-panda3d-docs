CPT ( TransformState  )

This is used to support NodePath::calc_tight_bounds(). It is not intended to be called directly, and it has nothing to do with the normal Panda bounding-volume computation.

If the node contains any geometry, this updates min_point and max_point to enclose its bounding box. found_any is to be set true if the node has any geometry at all, or left alone if it has none. This method may be called over several nodes, so it may enter with min_point, max_point, and found_any already set. Adds the node's contents to the CullResult we are building up during the cull traversal, so that it will be drawn at render time. For most nodes other than GeomNodes, this is a do-nothing operation.
