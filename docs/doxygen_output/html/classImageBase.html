Detailed Description

This specialization of ProgramBase is intended for programs that read and/or write a single image file. (See ImageMultiBase for programs that operate on multiple image files at once.)

This is just a base class; see ImageReader, ImageWriter, or ImageFilter according to your particular I/O needs.

◆ post_command_line()

bool ImageBase::post_command_line ( )
protectedvirtual

This is called after the command line has been completely processed, and it gives the program a chance to do some last-minute processing and validation of the options and arguments. It should return true if everything is fine, false if there is an error.

Reimplemented from ProgramBase.
