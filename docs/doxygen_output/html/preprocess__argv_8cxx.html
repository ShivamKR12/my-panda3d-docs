void preprocess_argv ( int &  argc,
char **&  argv 
)

Processes the argc, argv pair as needed before passing it to getopt(). If this program is running on Windows, but not within Cygwin, this ignores the incoming argv, argv values, replacing them from the GetCommandLine() string, and expanding glob patterns like *.egg to a list of all matching egg files. On other platforms, this function does nothing and returns argc, argv unchanged.

The argc and argv values are modified by this function, if necessary, to point to statically-allocated memory that will be valid until the next call to preprocess_argv().
