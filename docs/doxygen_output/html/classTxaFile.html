Detailed Description

This represents the .txa file (usually textures.txa) that contains the user instructions for resizing, grouping, etc. the various textures.

Member Function Documentation

bool TxaFile::match_egg ( EggFile *  egg_file) const

Searches for a matching line in the .txa file for the given egg file and applies its specifications. If a match is found, returns true; otherwise, returns false. Also returns false if all the matching lines for the egg file include the keyword "cont".
