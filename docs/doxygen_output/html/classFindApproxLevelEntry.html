Detailed Description

This class is local to this package only; it doesn't get exported. It represents a single node under consideration for matching at a single point in the breadth-first search.

Considers the node represented by the entry for matching the find path. If a solution is found, it is added to result; if the children of this node should be considered, the appropriate entries are added to next_level.

The return value is true if result now contains max_matches solutions, or false if we should keep looking.
