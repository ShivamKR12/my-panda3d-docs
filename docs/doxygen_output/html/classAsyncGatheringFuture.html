Detailed Description

Specific future that collects the results of several futures.

Member Function Documentation

◆ cancel()

bool AsyncGatheringFuture::cancel ( )
overridevirtual

Cancels all the futures. Returns true if any futures were cancelled. Makes sure that all the futures finish before this one is marked done, in order to maintain the guarantee that calling result() is safe when done() returns true.
