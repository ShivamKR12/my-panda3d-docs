Detailed Description

template<class T>
class WeakPointerTo< T >

WeakPointerTo is similar to PointerTo, except that it does not actually prevent the referenced pointer from deleting. Instead, the referenced pointer is allowed to delete, but if this happens then was_deleted() will return true, and it will be an assertion error to dereference the pointer thereafter.
