Detailed Description

Converts an egg data structure, possibly read from an egg file but not necessarily, into a scene graph suitable for rendering.

This class isn't exported from this package.

◆ start_sequences()

void EggLoader::start_sequences ( )

Starts all of the SequenceNodes we created looping. We have to wait until the entire graph is built up to do this, because the SequenceNode needs its full set of children before it can know how many frames to loop.
