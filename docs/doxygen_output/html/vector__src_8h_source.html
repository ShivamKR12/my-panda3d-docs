15 * This file defines the interface to declare and export from the DLL an STL
16 * vector of some type. To use this file you must #define a number of symbols
17 * and then #include it from a .h file. You also must do the same thing with
18 * vector_something_src.cxx from a .cxx file. This is necessary because of
19 * the complexity involved in exporting a vector class from a DLL. If we are
20 * using the Dinkumware STL implementation, it is even more complex. However,

22 * libraries are able to export symbols (including templates) without any
