◆ PTA_LMatrix4d

typedef PointerToArray<UnalignedLMatrix4d> PTA_LMatrix4d

A pta of LMatrix4ds. This class is defined once here, and exported to PANDA.DLL; other packages that want to use a pta of this type (whether they need to export it or not) should include this header file, rather than defining the pta again.

We actually wrap UnalignedLMatrix4d, in case we are building with SSE2 and LMatrix4d requires strict alignment.
