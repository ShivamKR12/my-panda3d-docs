Detailed Description

bool RangeDescription::parse_parameter ( const std::string &  param)

Parses a string of comma- and hyphen-delimited unicode values, in decimal and/or hex, including possible bracket-delimited ASCII characters, as may have been passed on a command line. Returns true if the parameter is parsed correctly, false otherwise.
