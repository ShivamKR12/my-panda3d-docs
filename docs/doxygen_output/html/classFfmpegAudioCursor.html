Read audio samples from the stream. N is the number of samples you wish to read. Your buffer must be equal in size to N * channels. Multiple-channel audio will be interleaved.

◆ reload_buffer()

bool FfmpegAudioCursor::reload_buffer ( )
protected

Reloads the audio buffer by decoding audio packets until one of those audio packets finally yields some samples. If we encounter the end of the stream, we synthesize silence.
